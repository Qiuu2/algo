## 0. 为什么先做这一步（不预判）

CTO 硬约束 (1) 明令：**Re 口径必须先举证**，unit 与 channel（串联翻倍）的层级写清，否则不喂 Re 进模型。
原因：T/S 参数（SPLo / BL / Re / Cms / Mms / ...）测的是 **单只拆机单元**；但产品的功放通道 **一路串联驱动两只单元**。
若把 unit-level 的 Re 误当 channel-level（或反之），灵敏度换算会整体偏 ~6 dB（电压/电流/功率参考混淆）。
**本文件不算 SPL**，只把 Re 层级钉死、把建模后果（电压如何分、声学如何加）写成可被 critic 逐条裁的口径声明。

## 2. 为什么 LEAP Revc 7.600 是 unit-level（送测语境论证）

- **送测对象 = 单只拆机单元**：`full_teardown_v2.md:48`「**拆机单元送测中**，预计 2–4 周回报」——实验室收到并测量的是**一只拆下来的单元**，不是整柱、不是串联对。
- **LEAP 面板自洽支持单只**：`KB-DRV-TEST-001_extracted.md:18-19, 33-34` — LEAP-4 双曲线 Delta Mass 法，由**阻抗扫描 + 加质量**拟合。面板 Qts = Qms·Qes/(Qms+Qes) = 4.625·2.140/6.765 = 1.463 ✓ 自洽（`:33, :53`）；Fs 自洽核 Fo=1/(2π√(Mms·Cms))=235.91Hz ✓（`:34`）。这些都是**单只驱动器**的机电参数，串联对不会给出这组自洽的单只 T/S。
- **数值互证锁单只**：LEAP Revc **7.600** ≈ 万用表单只 DC **7.4**（差 0.2Ω / 2.7%，仪器间正常），二者**都是单只音圈直流电阻 Re**（`KB-DRV-TEST-001_extracted.md:87-90`）。若 Revc 是 channel-level（串联对），其值应 ~15Ω，与 7.4Ω 单只 DC 矛盾。**实测 7.600 ≈ 7.4 → 必为 unit-level**。
- **结论**：**Revc 7.600 = unit-level**（同 §1 (i)(iii)），与整套 T/S（SPLo/BL/Cms/Mms/Fs/Qts...）同为 **unit-level**。

## 3. 一致性核（单只 ~7.4–7.6，通道 ~14.8–15，层级 CLEAN）

- **桥接核**：2 × 7.4 = **14.8** ≈ **15**（串联 DC，测量散差 ~1.3%/0.2Ω 内）。
  → 单只两两串联的预期 DC（14.8）与独立测得的通道串联 DC（15）一致。
- **层级 CLEAN（无交叉污染）**：
  - **unit-level** ≈ **7.4–7.6 Ω**（万用表单只 DC 7.4 / LEAP Revc 7.600，双轨 [L1]）。
  - **channel-level** ≈ **14.8–15 Ω**（2×单只 推算 14.8 / 万用表串联直测 15，双轨 [L1]）。
  - 两层级数值不重叠、互不冒充，且通过 ×2 桥接互证 → **层级干净**。
- 全部为 **[L1]**（万用表实测 + LEAP 仪器）；铭牌 8Ω [L4] 不参与。

## 4. 建模后果（精确写明 —— 这是喂模型前必须钉死的口径）

### 4a. 参数层级
- **T/S 参数（SPLo / BL / Re / Cms / Mms / Fs / Qts / Sd / Levc / ...）全部是 unit-level**（单只拆机单元，§1–§2）。
- 喂电声模型的 **Re = unit-level 7.4–7.6 Ω**（取 LEAP Revc 7.600 为主，万用表 7.4 为双轨核）。**严禁**把 15Ω 当 Re 喂单只模型。

### 4b. 拓扑（功放通道 → 两只单元串联）
- 拓扑 [L1 拆机]：**16 只单元，8 路功放**；通道 c 驱动对 {c, 15−c}，**两只单元串联**（pair DC ~15Ω = 2×7.4 ✓）。
  - 来源：`定向音柱AI数据_含追问回复_extracted.md:12`（A 区第 n + B 区第 (17−n) 串联，8 路，d=55mm）。
  - 配对/权重与 DSP 侧一致：`sprint4/dsp/fira/dolph_w8_q15.csv` —— 8 行，pair 列显式 `{0,15} {1,14} ... {7,8}`（即 {c,15−c}），Dolph −20dB 通道权重 w8（w_float 0.50431…→1.0，w_q15 16525…→32768）。
- **串联电路后果**：一路串联 → **两只单元电流相同（同一 i）**；通道端电压 **大致各分一半**（两只 Re 近似相等，~7.4Ω each，故每只 ~½ 通道电压；严格分压另含各自感抗/动生反电势，建模时按串联同电流处理最干净）。
- **声学叠加后果**：对 {c, 15−c} 同电流 → 两只声输出**正轴向相干相加**（同相驱动、broadside）。**正轴向 on-axis 两只单元声压相干叠加 → +6 dB（电压同相、相干）**，这一项必须显式记入系统灵敏度。

### 4c. 参考约定必须声明（喂模型/出结果时强制带）
任何「功率/电压灵敏度」数字**必须声明其参考基准**，否则口径不清：
- **per-unit 1 W**？（单只 1W 输入，单只 Re=7.6Ω）
- **per-channel 1 W**？（一路串联对 1W，通道 Re≈15Ω，功率两只均分）
- **total 1 W**？（整柱 8 路合计 1W）
- **2.83 V**？（铭牌惯例 2.83V/1m，对 8Ω 标称 = 1W；但实测 Re=7.6Ω/通道 15Ω，2.83V 对应的实际功率 ≠ 1W，须按实测 Re 重算）
> LEAP SPLo 82.161 dB 的参考本身 = 「LEAP reference sensitivity（1W/1m 等效）」，但其**绝对电平条件未定** —— 测量距离/电压/障板/平滑截图未标（`KB-DRV-TEST-001_extracted.md:11-12, 37, 62-63, 139`）→ **SPLo 绝对电平 = [L2-条件未定]，本证据包原样承袭此 caveat**，不得当作已坐实的 1W/1m 绝对值。形状 [L1]，绝对电平 [L2-条件未定]。
