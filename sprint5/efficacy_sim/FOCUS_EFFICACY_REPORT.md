## 1. 一句话物理结论

**v1 在轴聚焦的有效性由近场瑞利极限 `R = 2L²/λ`（L=0.825m，c=343）严格门控：聚焦只在「焦距 F < R」即仍处近场时物理上真实存在；一旦 F ≥ R 退化为远场，聚焦塌缩为 broadside，焦点增益归零。** 结论是**频率越高、距离越近，聚焦越实**：500Hz 在所有产品距离上聚焦几乎为空（全距离焦点增益 ≤ 0.075 dB）；可观的焦点增益（约 4.8–8.1 dB）与跨区「聚焦超额」隔离（最高约 3.9 dB）只出现在**高频（4k/6kHz）+ 近距（2–3m）**。

### 跨区隔离（深度分区核心指标）[L2/仿真, 双轨]

「表观隔离」`isolation_z1_on-off` 几乎全部来自 1/r 几何扩散（近区天然更响），**不是聚焦能力**。能体现聚焦真实贡献的是**聚焦超额** `focusing_excess = 表观隔离 − 几何扩散`：

**读法**：分区若只靠几何扩散，那不是「定向音柱聚焦」带来的，普通点声源也有。真正归因于在轴聚焦/深度门控的「净」隔离能力 = 聚焦超额，**最高约 3.9 dB（6kHz, 2m/5m），低频近乎 0**。

## 3. 场景映射（物理支持哪些分区，不支持哪些）

| 场景 | 距离 | 频段 vs 近场极限 | 物理判读（仅指标，不是裁决） |
|---|---|---|---|
| **博物馆讲解** | ~2–5m | 2–5m 处：1k 近场到 3m，2k 到 ~8m，4k/6k 全覆盖 | **最有利场景**。近距 + 可用频段重叠最大。4k@2m=4.85dB、6k@2m=8.09dB 焦点增益；2/5m 聚焦超额 6kHz 达 3.9dB。**但讲解语音主能量在 0.3–3kHz，该带焦点增益与聚焦超额都很小（1k@3m=0.135dB，2k 聚焦超额 ≤0.5dB）**——高频好看的数字落不到语音主带。 |
| **商场分区广播** | ~3–8m | 3m 处 1k 临界、2k 仍近场；8m 处仅 4k/6k 仍近场 | **部分支持，且只在高频近端**。3m 区高频可用（4k@3m=2.19dB，6k@3m=4.92dB）；8m 区低于 2kHz 全部 `void`。分区净隔离（聚焦超额 3m/8m）最高 1.9dB@6k，语音主带 ≤0.2dB。 |
| **车站广播** | ~5–15m | 15m 远超绝大多数频段瑞利极限（500Hz R=1.98m、1k=3.97m、2k=7.94m，仅 4k=15.87m/6k=23.81m 勉强够 12m） | **基本不支持深度分区**。8m 区仅 2k 以上近场，12–15m 区仅 4k/6k 近场且焦点增益已掉到 0.14–0.31dB。对宽带广播语音（含低频），15m 处在轴聚焦**物理上为空**——明说：车站远区的「深度分区」premise 在本几何上不成立。 |

> 跨场景共性：**所有可观焦点增益都集中在「语音主能量带之外的高频 + 近距」**；在 0.3–3kHz 语音核心带内，全部距离的焦点增益与聚焦超额都很小。这是 CTO 判断 v1 聚焦 premise 时必须正视的频带错配。

## 4. 诚实边界（本仿真**不**覆盖）

1. **点声源各向同性阵元**：未建模阵元自身指向性 / 衍射 / 互耦；≥2kHz 真实阵元有指向性，故本仿真**低估**真实高频指向性 —— 高频焦点增益趋势是 **L2 方向性，不是验收数字**。
2. **房间声学 / 混响 / 反射 / 边界**：全自由场。真实博物馆/车站/商场有强反射与混响，会显著侵蚀近场聚焦带来的隔离（混响场近似各向同性，吃掉聚焦超额）。
3. **真实喇叭响应**：未建模换能器频响 / 相位 / 灵敏度 / 失真；仅相对 dB，**绝对 SPL 未建模**。
4. **频带上界**：截至 6kHz；≥6236Hz 栅瓣区已排除（与锁定基线一致）。
5. **仅对称激励**：离轴/横向 steering 物理不可实现（拓扑约束 + DEC-S3-DSP-03），未仿真。
6. **L2→L1 缺口**：全部为仿真，**无任何实测**。任何进入选型/承诺/PRD 的聚焦效能数字，须经消声室 / 现场实测（anechoic + in-situ）升级到 L1 后方可作不可逆依据。

## 5. 判定留白（仅指标 — 「够不够格」是 CTO/PRD 的裁决）

本报告**不下效能裁决**。要把这些指标转成 v1 聚焦 premise 的 go/no-go，CTO/PRD 须先回答以下阈值问题：

1. **分区最小有用隔离 (dB)**：深度分区认定「有效」需要多少**净聚焦超额** dB？（本仿真：自由场最高约 3.9dB@6k 近距；语音主带 ≤0.5dB。）若 PRD 阈值 > ~4dB，则自由场已不达标，混响后更低。
2. **目标语音频带**：聚焦效能是否必须落在语音主能量带（~0.3–3kHz）？若是，则本几何在该带内聚焦近乎为空，高频数字无法救场。
3. **目标工作距离**：分区距离上限是多少？若含车站 12–15m 宽带场景，物理上超近场极限 → premise 不成立，应在 PRD 明确剔除或改约束。
4. **隔离 vs 几何扩散归因**：分区 KPI 是否扣除 1/r 几何扩散后只认聚焦净贡献？（强烈建议如此，否则会把普通点声源的扩散误记为聚焦能力。）
5. **混响余量**：自由场聚焦超额需预留多少混响损失余量，才能在真实场景达标？（需 L1 现场实测标定。）
