## (a) Sprint 3 目标与范围

**主目标**：将已锁定的算法/硬件基线从"纸面可行"推进到"EZKIT 实测可行 + 物理样机验证"。

| 范围项 | 内容 | 验收标准 |
|--------|------|----------|
| **S3-G1 树形 FIR C 实现** | 在 CCES 落 dyadic 树形半带结构 C 代码（替代全速率参考实现） | EZKIT 实测 MCPS ≤ 峰值的 1/10（即 ≥10× 裕量）；闭环 R1 |
| **S3-G2 定点化** | 浮点 → Q31 定点移植，子带 FIR + DAS 波束 | 定点 vs 浮点偏差 ≤ 1e-10（线性部分）；阻带 ≥55dB 保持 |
| **S3-G3 群延迟优化** | 端到端处理延迟实测与降延迟 | 端到端 ≤5ms（广播上限）；闭环 R2 |
| **S3-G4 扬声器单元选型** | 选定并送样阵元扬声器单元（小口径全频/高音单元） | 单元规格满足 1k-8kHz 频响 + 目标灵敏度；支撑 R3 |
| **S3-G5 声学实测对标** | d=55mm 实阵指向性 + 绝对 SPL 实测 | 仿真 vs 实测偏差 ≤±3dB；对标竞品 0° 106-111dB；闭环 R3 |

| **S3-G7 单板原理图/PCB v0.1** | ADSP-21569 + ADAU1966A + TAS5825M×8 主板设计 | 通过 ERC/DRC；DSP 散热预留（小散热片+开孔） |

## (e) R1–R4 在 Sprint 3 的闭环路径

| 风险 | 等级 | 闭环任务 | 负责 Agent | 闭环判据 | 失败处置 |
|------|------|----------|-----------|----------|----------|
| **R1** | P0 | dyadic 树形 FIR 落 C + EZKIT 实测 MCPS（S3-G1） | DSP | 实测算力裕量 ≥10× | 算法降阶 / 升级 CTO（ESCALATION） |
| **R2** | 高 | 端到端群延迟实测 + 降延迟优化（S3-G3） | DSP + 测试 | 端到端 ≤5ms | 缩短 FIR 阶数 / 减缓冲 / 上报权衡 |
| **R3** | 中 | 扬声器单元选型 + 消声室 SPL 实测（S3-G4/G5） | 硬件 + 声学 + 测试 | 0° SPL 对标竞品 106-111dB，偏差可接受 | 换单元 / 调整功率预算 |
| **R4** | ~~验证项~~ **CLOSED** | ~~d=30mm 重算 BW@2k~~ → MOOT：基线统一 d=55mm（DEC-S3-GEOM-01），d=30 BW@2k 重算前提作废；BW@2kHz 验证并入 R5（d=55 主路线唯一 BW 风险） | 声学 | — | — |

**闭环顺序建议**：
1. **并行启动**：R1（DSP 落 C，软件链）+ R5（声学 d=55mm BW@2k 仿真，仿真链）— 互不依赖，最先可动。（原 R4 d=30 重算已 MOOT/CLOSED。）
2. **R1 通过后** → R2（同一 EZKIT 实测延迟）。
3. **扬声器送样 + 消声室档期到位后** → R3 + R5 实测复核（同一次消声室测量批次完成，d=55mm 实阵）。
4. **R1/R2/R5 全闭环** → 触发 Gate 3，CTO 审定进入 PCB 打样/样机阶段。
