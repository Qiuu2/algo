## 2. 供应商联络模板（扬声器单元）

> 用途：发给候选喇叭单元供应商（建议 ≥3 家），索取完整 T/S 参数与一致性指标 + 样品。
> 这一步直接决定 R3（绝对 SPL 盲区）与 DEC-S2-013（1kHz 一致性要求）能否闭环。

```
主题：定向音柱项目 — 全频单元 datasheet 与样品索取

贵司您好：

我司正在研发定向音柱（线阵）产品，需选型小口径全频扬声器单元，
拟向贵司索取候选型号的完整规格与样品。需求如下：

【应用】16 单元线阵，单元间距 55mm（拆机实测基线，DEC-S3-GEOM-01），阵列总长约 825mm，
        工作频段 500Hz–8kHz（强指向频段 ≥1kHz；强指向规格上限按方向1降级为 6kHz 对内/5kHz 对外），
        每单元额定 10W RMS / 8Ω。

【必须提供的参数（datasheet 或测试报告）】
  □ 灵敏度 Sensitivity（dB SPL @ 2.83V/1m）        ← 决定整机 SPL，最关键
  □ 额定/峰值功率 (W RMS / W peak)
  □ 阻抗 Z（标称 8Ω）与直流电阻 Re (Ω)
  □ 力系数 BL (T·m)
  □ T/S 参数：Fs(谐振频率)、Qts、Qes、Qms、Vas、Mms、Cms、Sd
  □ 频率响应曲线（含 1k–8kHz 段）
  □ 失真 THD（典型功率下）

【一致性指标（量产关键，务必提供）】
  □ 同批次单元的灵敏度一致性 (± dB)
  □ 同批次单元的相位/阻抗一致性
  □ 量产幅相一致性可控范围（目标：幅度 ≤±2dB，相位 ≤±21° @1kHz）
     —— 我方 1kHz 指向性优化对一致性敏感，请说明贵司可达到的等级

【样品】每候选型号 ≥16 只，用于消声室阵列实测（费用/周期请告知）。

【商务】1k 台/年量级的参考单价与 MOQ。

期待回复，谢谢！
```

**候选供应商方向（参考，待核实）**：国产音柱专用全频单元厂（如赛微、通泰、歌尔系/国光系 OEM 等），优先选**能提供一致性分选/配对服务**的厂家。
