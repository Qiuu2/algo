## 1. 项目一句话定义 + 目标场景

> **一句话**：基于**相控阵线阵（DAS + 4 子带波束形成）**、跑在 **ADI SHARC+ ADSP-21569** 上的**定向音柱**，把声音定向投送到指定区域。

| 项 | 内容 |
|----|------|
| 产品 | 定向音柱（Column Speaker），强指向性扬声器阵列 |
| 目标场景 | 博物馆讲解 / 车站广播 / 商场分区广播（**固定前向安装，无电子偏转需求**）|
| 播放方向 | **单向播放**（无对话回路、无唇同步硬约束）→ 延迟规格得以放宽至 <30ms |
| 指向性需求 | **仅 ≥1kHz 要求强指向性**；250–500Hz 不作指向性要求（DEC-S1-002）|
| 核心指标 | -6dB 波束宽度（2kHz 目标 ≤30°）、旁瓣 -20dB、栅瓣安全频率 |

### 3.2 单一几何基线（d=55 统一，DEC-S3-GEOM-01）

> **2026-05-29 PF-8 修正**：撤销"两条几何并存"。自研基线与竞品工程基准**统一为 d=55**——d=30 系 Sprint2 视觉估测错误值，拆机实测真值为 d=55（DEC-S3-003）。统一决策由来见 **DEC-S3-GEOM-01**（决策日志），受 CTO 硬约束 **SC-S3-GEOM-01** 永久边界约束。

| 维度 | 统一自研基线 = 竞品工程基准（d=55）|
|------|----------------------------------|
| 来源 | 拆机 L1 实测真值（DEC-S3-003）；统一决策 DEC-S3-GEOM-01 |
| 阵元数 N | 16 |
| 间距 d | **55mm**（L1 实测）|
| 孔径 L | 825mm |
| 驱动 | **8 路 A/B 对称串联，broadside-only（无电子偏转）** |
| 加权 | Dolph-Chebyshev -20dB |
| 用途 | 自研量产路线 + 快速验证线（同几何，统一）|
| BW@2kHz（全角-6dB）| **14.5° [L2]**（大孔径，余量极大）|
| BW@1kHz（全角-6dB）| **29.28° [L2]**（压线达标，0.72° 余量 → R5 主路线唯一 BW 风险）|
| 栅瓣安全 | 500Hz–5k 安全；**6.2–8kHz 有栅瓣**（半波长判据 3118Hz / 严格判据 6236Hz [L1/几何]，R6 规格降级管理）|

> ⚠️ **栅瓣边界**：d=55 在 6.2–8kHz 有栅瓣，强指向上限按规格降级 6k（对内）/5k（对外）；6–8kHz 强指向需求受 SC-S3-GEOM-01 永久边界约束（R7，触发 d 重议）。

> 🗄️ **历史 d=30 数字（归档，不删，不可作决策依据）**：基于**错误几何 PF-8** 的 d=30 数字——BW@2k=26.8°、栅瓣临界 5.7kHz、超指向 D/B 全套 ε/WNG/容差（PF-7）——均**作废**，仅作历史留痕，**不可作任何决策依据**。引用波束/栅瓣数据一律以 d=55 为准。

### 3.3 DSP / 系统基线

| **① 半角误读** | BW 必须是**全角-6dB = 2×单边半角** | AC-WP01 初版把单边半角 14.63° 当成 BW@1k，并据此**误判 numpy 有"因子-2 bug"**、一度拟撤销 1kHz 超指向。经 DSP 审计 + 主 Claude MATLAB 复算 + Critic 独立阵因子复算**三方推翻**：正确全角=29.28°（=numpy 29.3°），numpy 无 bug；根因是 MATLAB `max()` 选中 ±180° 端瓣。物理铁证：-6dB 全角 > -3dB 全角。| 全项目统一口径 **BW=全角-6dB**；AC 报告用 `beamwidth()` 第一返回值；numpy 补"峰必在 0°"+ "-3dB 全角下界" sanity check（DEC-S3-DSP-06，memory `verify-teammate-numbers-before-acting`）|

| **④ 估测当实测** | 视觉目测值 ≠ L1 实测；实测与 LOCKED 冲突须重审，不可"并存"折衷 | 阵元间距 **d=30mm** 系 Sprint2 视觉/目测估值（无卡尺），LOCKED 进 DEC-S2-006；Sprint3 拆机实测真值 **d=55mm** 与之冲突（孔径差 1.83×），却未触发强制重审，反被"两条几何并存"合理化掩盖，延至 **PF-8** 才以 DEC-S3-GEOM-01 撤销 d=30（PM 在 PF-8 启动前已先以判据校准纠正 3.1k/6.2k 误读，防新假性完成）。| **POLICY-PROV-001 v1.2**：新增 **L0 等级**（纯目测，严禁任何 LOCKED 决策）+ **L1↔LOCKED 冲突强制重审规则**（禁"并存"折衷）+ **Critic C6 几何门禁**（几何/尺寸 LOCKED 前必须 L1 实测，否则 BLOCKER）；CTO 硬约束 **SC-S3-GEOM-01**（PRD 不新增 6–8kHz 强指向硬需求为前提，否则触发 d 重议）|
