## 0. 问题陈述

锁定方案在 1kHz 的 −6dB 波束宽度 = **55.2°**（来自 sweep_results.csv 仿真值），
明显宽于竞品反推的 **36.6°**。CTO 要求 ≥1kHz 强指向，55° 在 1kHz 偏弱。

**物理根因**：N=16/d=30 阵在 1kHz 的总孔径 L=450mm，λ=343mm，**L/λ=1.31**。
常规相控阵 −6dB BW 受瑞利衍射极限约束（≈0.886·λ/L），Dolph−20dB 加权为压旁瓣进一步展宽主瓣，
故得 55.2°。**纯相控阵在 1kHz 物理上无法显著突破 ~40°**——这正是差分/超指向方案的切入点。

## 1. 差分波束形成仿真（D-2 核心）

### 1.1 方法

差分波束形成（Differential Beamforming）= 扬声器阵的超指向（superdirectivity）。
本质是用相邻阵元**幅度反相组合**，在各向同性噪声场下等效 MVDR 加权：

```
w = (Γ + εI)⁻¹·a / [aᴴ·(Γ + εI)⁻¹·a]
Γ_ij = sinc(k·d·|i−j|)   （各向同性噪声相干矩阵）
a = broadside 导向矢量；ε = 正则化（控制超指向程度 ↔ 鲁棒性）
```

理想 N 阶差分端射方向图为 cos^N 型（频率无关）。

### 1.2 各阶差分方向图指标（理想 cos^N 端射型）

| 阶数 | −6dB BW | 前后比* | 旁瓣 |
|------|---------|---------|------|
| 1阶 (cardioid) | 179.7° | ≤30 dB | −25 dB |
| 2阶 | 130.8° | ≤30 dB | −25 dB |
| 3阶 | 107.9° | ≤30 dB | −25 dB |

> \* 理想 cardioid 后向为数学零点（−∞），实际受阵元误差/WNG 限制，封顶约 25~30dB。
> 注：表中为差分阵**端射**主瓣宽度（沿阵轴方向），方向图本身频率无关；
> 用于 broadside 音柱时通过超指向加权在 broadside 方向收窄（见 §3）。

### 1.3 ★核心成本：白噪声增益 WNG（必须如实给出）

WNG 衡量阵列对阵元自噪声/幅相误差的抑制能力，**越负越差**。
差分阵 WNG ∝ (kd)^(2N)，小孔径低频急剧恶化：

| 阶数 | WNG@500Hz | **WNG@1kHz** | WNG@1.5kHz |
|------|-----------|--------------|------------|
| 1阶 | −8.3 dB | **−2.6 dB** | +0.3 dB |
| 2阶 | −19.7 dB | **−8.3 dB** | −2.2 dB |
| 3阶 | −30.6 dB | **−13.4 dB** | −4.3 dB |

**解读**：
- 1阶在 1kHz WNG=−2.6dB 尚可接受；
- 2阶 WNG=−8.3dB 已显著（灵敏度损失约 8dB，自噪声放大）；
- **3阶 @500Hz WNG=−30.6dB 是灾难性的**——意味着阵列自噪声/误差被放大 1000 倍，工程不可用。

### 1.4 阵元幅相一致性容差需求（差分方案的隐性成本）

要保证差分波束有效（误差导致方向图劣化 < 3dB），阵元须满足：

| 阶数 | WNG@1kHz | 允许误差功率 | **增益容差** | **相位容差** |
|------|----------|-------------|-------------|-------------|
| 1阶 | −2.6 dB | −5.6 dB | ±3.21 dB | ±21.2° |
| 2阶 | −8.3 dB | −11.3 dB | ±1.68 dB | ±11.1° |
| 3阶 | −13.4 dB | −16.4 dB | **±0.93 dB** | **±6.1°** |

**关键警示**：3阶差分要求阵元增益匹配 ±0.93dB、相位匹配 ±6.1°——
对消费级扬声器单元（典型一致性 ±1~2dB / ±10~15°）**几乎不可量产达标**，
且温度/老化漂移会进一步破坏匹配。这是差分高阶方案的**致命成本**。

## 2. 混合方案 BW@1k 优化结果

「差分 + 相控混合」= 1kHz 用超指向加权（等效低阶差分），≥1.5kHz 退回 Dolph 相控。
通过正则化 ε 扫描，得 BW-WNG 权衡曲线：

| 正则化 ε | BW@1k | WNG@1k | 增益容差 | 相位容差 | 评价 |
|---------|-------|--------|---------|---------|------|
| 0.3 | 42.2° | +9.3 dB | ±12.7 dB | ±83.8° | 鲁棒，接近常规 |
| 0.1 | 39.0° | +6.1 dB | ±8.8 dB | ±58.1° | 鲁棒 |
| 0.03 | 36.5° | +3.6 dB | ±6.6 dB | ±43.6° | **达竞品水平，仍鲁棒** |
| 0.01 | 35.4° | +2.0 dB | ±5.5 dB | ±36.2° | **达竞品水平，鲁棒** |
| 0.003 | 34.6° | −2.1 dB | ±3.4 dB | ±22.5° | 开始变脆 |

**优化后 BW@1k 目标值对比**：

| 方案 | BW@1k | WNG@1k | 说明 |
|------|-------|--------|------|
| 现状 Dolph 相控 | **55.2°** | (基准) | 偏宽 |
| 竞品反推 | **36.6°** | — | 目标参考 |
| **混合-保守 (ε=0.01)** | **35.4°** | **+2.0 dB** | **推荐工作点** |
| 混合-激进 (ε=0.003) | 34.6° | −2.1 dB | 不必要的冒险 |

**重要发现**：达到竞品 36.6° 水平**不需要灾难性的超指向**——
ε=0.01~0.03 即可把 BW 从 55°收到 35~36°，且 **WNG 仍为正（+2~3.6dB，无效率损失）**，
容差需求 ±5~6dB/±36~44°（远比 3阶纯差分宽松）。这说明 16 元 30mm 阵在 1kHz
做"温和超指向"是可行的，不必上高阶差分。

> 图见 `diff_1khz_comparison.png`（6 子图：差分方向图 / 现状vs混合 / WNG-频率 / BW柱状 / 容差 / WNG柱状）。

## 3. 增量估算

### 3.1 算力增量

差分/超指向补偿 FIR 在低速率子带运行（SB0 抽取至 ~3kHz，SB1 低端 ~6kHz），16 通道，每通道 32 抽头：

| 子带 | 速率 | 配置 | 算力 |
|------|------|------|------|
| SB0 (500–1k) | 3 kHz | 16ch × 32tap | 1.54 MMAC/s |
| SB1 低端 | 6 kHz | 16ch × 32tap | 3.07 MMAC/s |
| **合计** | — | — | **≈ 4.61 MMAC/s** |

相对 SHARC+ 1GHz（~数千 MMAC/s 量级）算力增量极小（<0.5%）。⚠️ **需 DSP agent 复核**。

### 3.2 延迟增量

| 子带 | FIR 群延迟 | 备注 |
|------|-----------|------|
| SB0 | ≈ 5.33 ms | 低速率→延迟较大 |
| SB1 低端 | ≈ 2.67 ms | |

⚠️ 子带分析-重建（滤波器组）的总延迟须由 **DSP agent 复核**；语音广播对 5ms 量级延迟不敏感，但需确认整体一致性。

### 3.3 复杂度增量

- **系数标定**：每台需逐通道幅相标定（差分对误差敏感），增加产线工序与测试设备；
- **温度/老化漂移**：扬声器 Re/灵敏度随温度变化，会破坏幅相匹配 → 超指向劣化（高阶尤甚）；
- **生产一致性**：保守混合(ε=0.01)要求 ±5dB/±36°，消费级单元可达标；高阶纯差分(±0.93dB/±6°)产线良率风险高；
- **EQ 复杂度**：超指向加权改变各通道幅频，需配套 EQ 重新整定。

## 4. ⚠️ Cost-vs-Benefit 裁决

### 4.1 收益侧
- BW@1k 从 55.2° → 35~36°，**达到/接近竞品水平**，满足"≥1kHz 强指向"诉求；
- 保守工作点(ε=0.01)收益实在：**−20° 主瓣收窄，且 WNG 仍 +2dB（无效率/鲁棒性代价）**。

### 4.2 成本侧
- 高阶纯差分(2/3阶)：WNG −8~−30dB、容差 ±0.9~1.7dB/±6~11°，**量产不可行、效率灾难**；
- 温度/老化漂移敏感性 → 长期一致性风险；
- 标定工序 + EQ 整定 + 延迟（需 DSP 复核）；
- 算力增量可忽略（4.61 MMAC/s）。

### 4.3 场景适配判断（CTO 场景：博物馆/车站/商场广播）

| 场景 | 1kHz 是否真需强指向 | 判断 |
|------|---------------------|------|
| 博物馆讲解 | 是（分区隔离、避免串扰） | 1kHz 是语音主能量区，强指向有价值 |
| 车站广播 | 部分（覆盖均匀度优先于强指向） | 强指向收益中等 |
| 商场分区 | 是（相邻区互不干扰） | 1kHz 强指向有价值 |

语音可懂度主能量在 0.5–3kHz，1kHz 恰在核心区，**适度强指向对分区隔离确有实际价值**。

### 4.4 明确建议：**有条件做**

✅ **推荐：采用"温和超指向"混合方案（ε≈0.01~0.03），BW@1k→35~36°，WNG 保持 +2~+3.6dB。**

**条件**：
1. **只做温和超指向，禁止 2 阶及以上纯差分**（WNG 灾难、量产不可行）；
2. 工作点 ε 设为可调参数，出厂按实测单元一致性整定（一致性差则提高 ε 牺牲少量 BW 换鲁棒）；
3. 增加产线幅相标定工序（±5dB/±36° 容差，消费级单元可达标）；
4. **延迟与算力增量须经 DSP agent 复核**后方可锁定；
5. 温度漂移做开环补偿或留足 ε 余量。

❌ **不推荐**：为追平竞品最后 1~2° 而把 ε 压到 0.003 以下（WNG 转负、鲁棒性骤降，收益微小、风险陡增）。

> **一句话裁决**：1kHz 收窄到竞品水平**值得做且代价可控**，但**必须走"温和超指向"路线**，
> 高阶差分是陷阱。最终落地前需 DSP agent 复核延迟/算力，硬件 agent 评估单元一致性与温漂。
