## 一、竞品阵列参数反推（任务 A）

### 1.1 方法论

采用线阵远场阵因子模型（点声源近似）：

```
AF(θ) = Σ w_n · exp(j·k·n·d·sinθ),  k = 2π·f/c,  c = 343 m/s
```

对 1kHz / 2kHz / 4kHz（CTO 指定强指向频段），在 0°/30°/60°/90° 四个实测角度，
以仿真-实测 RMS 误差为目标函数，在以下空间穷举搜索：
- N ∈ {8, 12, 16, 20, 24, 32}
- d ∈ {25, 28, 30, 32, 35, 40 mm}
- 加权 ∈ {均匀, Hanning, Dolph-20dB, Dolph-30dB}

> 技术说明：线阵阵因子 AF(θ) 满足 AF(0°) = AF(180°)（因 sin0°=sin180°=0），
> 故分析范围限制在 θ∈[0°, 90°]，以避免后向镜像峰干扰旁瓣分析。

### 1.2 反推结果

**最佳匹配参数**：
- 阵元数：**N = 20**
- 阵元间距：**d = 35 mm**
- 阵列长度：**L = 665 mm**
- 加权方式：**dolph20**
- 平均 RMS 误差：**2.84 dB**（3频率 × 4角度）

**Top-3 候选组合**：
```
  1. N=20, d=35mm, 加权=dolph20       RMS=2.84dB
  2. N=32, d=28mm, 加权=dolph20       RMS=3.36dB
  3. N=24, d=25mm, 加权=dolph20       RMS=3.62dB
```

### 1.3 竞品特征分析

### 1.4 不确定性评估

|-------------|------|-----------|

> **状态**：本节针对 Critic 退回的 2 个 🔴BLOCKER（F-AC-01 / F-AC-02）及 2 条 CTO 决策支撑需求做定向返工。

### 7.1 F-AC-01 修订：撤回 4 点插值的竞品 BW，改用反推模型隐含 BW

**问题确认（Critic 完全正确）**：
原报告 §1.3 给出的竞品 BW（1k≈16.0° / 2k≈14.9° / 4k≈19.1°）是基于实测 **4 个角度（0/30/60/90°）线性插值**得到的，
属于**方法学假象**。实测数据显示 0°→30° 之间已发生剧烈跌落：

| 频率 | 30° 处相对电平 | 结论 |
|------|---------------|------|
| 1kHz | −22.5 dB | 0°→30° 跌幅 22.5dB ≫ 6dB |
| 2kHz | −24.1 dB | 0°→30° 跌幅 24.1dB ≫ 6dB |
| 4kHz | −18.8 dB | 0°→30° 跌幅 18.8dB ≫ 6dB |

由于 −6dB 交点必然落在 **0°~30° 之间**，而该区间内**无任何实测采样点**，线性插值会把交点错误地外推到 14°~19°，
**严重高估**竞品真实主瓣宽度。

**🔴 撤回声明**：原 §1.3 表中"竞品 BW @1kHz=16.0°、@2kHz=14.9°、@4kHz=19.1°"三个数值**予以撤回**，
应视为**不可靠的上界估计**。仅凭 6 角度采样，**无法可靠估计竞品 −6dB 主瓣宽度**。

**改用反推模型推算"模型隐含 BW"（含不确定带）**：
基于反推最优模型（N=20，d=35mm，Dolph-20dB，RMS=2.84dB），并以 Top-3 反推候选给出不确定带：

| 频率 | 模型隐含 BW（中心） | 不确定带（Top-3 候选散布） |
|------|---------------------|---------------------------|
| 1kHz | 36.6° | 27.9° ~ 42.7° |
| 2kHz | 18.1° | 13.9° ~ 21.0° |
| 4kHz | 9.0° | 6.9° ~ 10.4° |

**拟合优度**：见新增图 `competitor_fit_quality.png`——将实测 6 点叠加到反推模型方向图上，
模型在 30°/60°/90° 实测点附近吻合良好（RMS=2.84dB），但**1kHz 实测 30° 处比模型方向图更低**，
提示竞品 1kHz 主瓣可能比纯阵因子模型更窄（阵元自身指向性 + 可能的 DSP 相位优化在低频也有贡献），
故 1kHz 模型隐含 BW（36.6°）应理解为**上界**。

> 方法学说明：模型隐含 BW 是"假设竞品就是该阵列配置"下的理论值，
> **不是实测值**；其可靠性依赖反推拟合优度，4kHz 因阵元指向性影响置信度为中。

### 7.3 栅瓣结论更正（CTO 决策支撑：d=30mm vs d=35mm）

**🔴 原 §3 "d=30mm 在整个工作频段均无栅瓣" 的表述不严谨，予以更正。**
该结论仅在 **≤4kHz** 成立；若考虑子带 SB3（4.9–8kHz）或全频 8kHz，结论不成立：

| 频率 | d=30mm：d/λ | d=30mm 状态 | d=35mm：d/λ | d=35mm 状态 |
|------|-------------|-------------|-------------|-------------|
| 4.0 kHz | 0.350 | 无栅瓣 | 0.408 | 无栅瓣 |
| 4.9 kHz | 0.429 | 无栅瓣 | **0.500** | **临界（旁瓣抬升）** |
| 6.0 kHz | 0.525 | **栅瓣** | 0.612 | **栅瓣** |
| 8.0 kHz | 0.700 | **栅瓣** | **0.816** | **栅瓣（±37.8°）** |

**更正要点**：
- 栅瓣条件为 **d/λ ≥ 0.5**（broadside 阵），并非仅看 4kHz。
- **d=35mm**：在 **4.9kHz 即达临界 d/λ=0.5**，SB3（4.9–8kHz）及 8kHz 确有**栅瓣抬升风险**（8kHz 栅瓣出现于 ±37.8°）。原"全频段无栅瓣"结论对 35mm **不成立，特此更正**。
- **d=30mm**：栅瓣起始频率约 **5.7kHz**，比 35mm 更高，对 SB3 高端更安全。

#### CTO 决策支撑：d=30mm 与 d=35mm 各一组候选（均衡档对比，N=24，Dolph-30dB）

| 指标 | d=30mm | d=35mm |
|------|--------|--------|
| 阵列长度 L | 690 mm | 805 mm |
| BW @2kHz | 见 §四（约 20.4°） | 17.8° |
| 栅瓣起始频率 | ≈5.7 kHz | ≈4.9 kHz |
| SB3(4.9-8kHz) 风险 | 较低 | **较高（需 DSP 子带限频或降采样）** |
| 指向性（2kHz） | 略弱 | 略强 |

> **决策建议（不拍板，供 CTO 参考）**：
> 若产品需覆盖 ≥4.9kHz 高频子带（SB3），**d=30mm 更稳健**（栅瓣裕量大 0.8kHz，阵列更短 115mm）；
> 若高频可由专用高频单元/子带限频处理、且追求 2kHz 更窄主瓣，**d=35mm 可接受**。
> CTO 当前规格重点在 ≥1kHz 强指向、2kHz≤30°，两者均达标；差异主要体现在 **SB3 栅瓣裕量**与**阵列长度**。
