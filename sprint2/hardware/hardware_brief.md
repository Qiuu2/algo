# 硬件设计简报 v0.1
**项目**：ITC 定向音柱（Column Speaker）  
**DSP 平台**：ADSP-21569（SHARC+ 1GHz，400-ball CSP_BGA 17×17mm）  
**音频通道**：16ch / 24ch 两版（同一硬件平台，扩展性设计）  
**日期**：2026-05-26  
**状态**：初版评估，待 Critic 审核

## A. 外围方案选型

### A.1 多通道 Codec / DAC

| 器件 | 型号 | 通道数 | 接口 | 关键参数 | 理由 |
|------|------|--------|------|----------|------|
| 16ch DAC（首选）| ADI **ADAU1966A** | 16ch | TDM（8xTDM16 或 2xTDM32） | 24bit，96kHz，SNR 107dB，I2C 控制，3.3V 单电源 | ADI 家族与 SHARC 原生兼容，无需额外胶合逻辑，ADI 有完整参考设计 |
| 8ch DAC（辅助/扩展）| ADI **ADAU1962A** | 8ch | TDM8/TDM16 | 24bit，192kHz，SNR 106dB | 2 片级联可凑 16ch；24ch 方案需 3 片（3×8）或 ADAU1966A×2 |
| 备选 16ch | TI **PCM9211×2** | 8ch×2 | I2S/TDM | 成熟货源，价格略低 | 跨家族需注意时序调试 |

**16ch 方案**：1× ADAU1966A（16ch TDM），BCLK=24.576MHz，FS=48kHz，32bit/slot，与 ADSP-21569 SPORT 直连，单路 TDM 搞定。  
**24ch 方案**：3× ADAU1962A（每片 8ch），使用 2 路 TDM16（SPORT0+SPORT1），DSP 侧需分配 2 个 SPORT 端口，DMA 配置增加但 SHARC 有充足 SPORT 资源。  
**影响汇总**：

| 项目 | 16ch | 24ch |
|------|------|------|
| Codec 数量 | 1× ADAU1966A | 3× ADAU1962A 或 2× ADAU1966A（共 16+16=32ch 取前24）|
| TDM 路数 | 1 路 TDM（SPORT0）| 2 路 TDM（SPORT0+SPORT1）|
| 功放板路数 | 16 路 | 24 路 |
| DSP 通道负担 | 16ch DAS | 24ch DAS（+50%，仍在 SHARC FAS 算力包络内）|

### A.2 D 类功放

**功率档位定义**（定向音柱，每单元扬声器额定 10W，峰值裕量 1.5×）：
- 每通道输出功率目标：**10W RMS @ 8Ω**（扬声器额定），D 类效率按 88% 估算

**多通道 Class-D 方案对比**：

| 型号 | 通道 / 片 | 电源电压 | 额定功率 | 效率 | 封装 | 适用档 |
|------|----------|----------|---------|------|------|--------|
| TI **TAS5825M** | 2ch | 4.5–26V | 2×30W(@8Ω) | ~90% | 36-VQFN | 16ch: 8片，成熟 |
| TI **TAS6424** | 4ch | 6–26V | 4×25W(@4Ω) | ~91% | 48-HTSSOP | 16ch: 4片，车规品质 |
| ADI **MAX9744** | 2ch | 4.5–14V | 2×20W | ~87% | 36-TQFN | 功率偏小，仅参考 |
| TI **TPA3255** | 2ch | 4.5–26V | 2×315W max | ~93% | 56-KDAP | 功率过大，不适合 |
| **推荐：TAS5825M** | 2ch/片 | 12–24V | 2×30W | ~90% | 36-VQFN | 综合最优 |

**推荐方案**：TI TAS5825M，每片 2 通道，16ch 需 **8 片**，24ch 需 **12 片**。
- 供电：24V @ 功放段，通过 I2C 进行音量/EQ 控制（亦可 bypass，由 SHARC 数字域处理后直接 I2S 输入功放）
- 每通道散热功耗（10W 输出）：10W × (1/0.9 − 1) ≈ **1.1W/通道**，16ch 合计约 **17.8W**，24ch 约 **26.7W**

### A.3 电源方案

#### 供电拓扑：AC-DC + 多路 DC-DC + LDO

```
AC 220V (市电)
    │
    ├─ [保险丝] → [共模扼流圈+X/Y电容 EMI Filter]
    │
    └─ [AC-DC 开关电源模块]  →  DC 24V Bus（主母线，20A，480W 峰值裕量）
            │
            ├─ [Buck 1: TPS543227 / LMR33630]   24V→12V @ 5A   (预留功放备用/风扇)
            │
            ├─ [Buck 2: TPS62085 / LMR16006]    24V→5V  @ 2A   (中间轨，系统逻辑)
            │      │
            │      ├─ [LDO 1: TPS7A4700]  5V→3.3V @ 500mA  (DSP IO/Codec DVDD)
            │      ├─ [LDO 2: TLV62568]   5V→1.1V @ 2A     (SHARC+ Core 1.1V)
            │      └─ [LDO 3: TPS62840]   5V→1.8V @ 500mA  (DSP 1.8V IO/PLL)
            │
            ├─ [Buck 3: TPS54360 低噪声]         24V→5VA @ 1A  (模拟域独立供电)
            │      └─ [LDO 4: TPS7A4700RGWR]  5VA→3.3VA @ 300mA  (Codec AVDD，噪声<4μVrms)
            │
            └─ [直接 24V]  → TAS5825M PVDD 功放供电（24V 直通）
```

**关键电源参数**：

| 电源轨 | 电压 | 器件（推荐） | 电流（16ch/24ch） | 备注 |
|--------|------|-------------|-----------------|------|
| SHARC+ Core | 1.1V | LDO TLV62568 | 1.5A / 1.5A | ADSP-21569 VDD_CORE |
| DSP I/O | 1.8V | LDO TPS62840 | 300mA | VDDIO_1P8 |
| 数字 I/O | 3.3V | LDO TPS7A4700 | 500mA | VDDIO，Codec DVDD |
| 模拟 | 3.3VA | TPS7A4700（低噪声） | 200mA | Codec AVDD，噪声敏感 |
| 功放 PVDD | 24V | 直通 DC Bus | 8A / 12A | TAS5825M PVDD |
| AC-DC 模块 | 24V | 明纬 MSP-450-24 或同类 | 20A 峰值 | 整机 480W 裕量 |

**上电时序**：1.1V Core → 1.8V → 3.3V → 3.3VA → 功放 PVDD（软启动）

### A.4 时钟方案

```
晶振 OSC1: 24.576MHz TCXO（Abracon ASTX-H12-24.576MHz 或 NDK NX3225SA）
    │
    ├─→ ADSP-21569 CLKIN（直接输入，内部 PLL ×N）
    │       DSP 内部 PLL: 24.576MHz × ~40 = 983.04MHz ≈ 1GHz
    │
    └─→ Codec MCLK（ADAU1966A / ADAU1962A MCLK 引脚）
            Codec 内部分频: MCLK/2 = 12.288MHz，FS = 12.288M/256 = 48kHz ✓
            BCLK = 24.576MHz，32bit×2ch×N slots = 满足 TDM 协议

时钟分配（如多片 Codec）:
    24.576MHz → 时钟缓冲器 TI CDCLVC1104（1:4 驱动，低抖动）
        ├─→ ADSP-21569 CLKIN
        ├─→ Codec_1 MCLK
        ├─→ Codec_2 MCLK（24ch 方案）
        └─→ Codec_3 MCLK（24ch 方案）
```

**时钟质量要求**：相位抖动 < 50ps RMS（满足 TDM 信号完整性），TCXO 频率精度 ±2.5ppm。

### C.2 三种工况下整机功耗估算

#### 假设基础参数
- 扬声器单元：8Ω，额定 10W（RMS），峰值 15W
- D 类效率：η = 90%（典型），待机 ICC < 5mA/片
- 平均语音内容功率 ≈ 额定的 10%（-10dB）；典型 -6dB 语音 ≈ 额定的 25%

| 工况 | 功放输出/ch (W) | 16ch 功放输入 (W) | 24ch 功放输入 (W) | DSP+Codec (W) | 电源损耗 (+15%) | **16ch 总计 (W)** | **24ch 总计 (W)** |
|------|----------------|-----------------|-----------------|---------------|----------------|-----------------|-----------------|
| **待机**（静音，DSP 空转） | 0 | 16×0.15=2.4 | 24×0.15=3.6 | 1.0 | +0.5 | **~4W** | **~7W** |
| **典型语音（-6dB）** | 2.5W | 2.5×16/0.9=44W | 2.5×24/0.9=67W | 2.6 | +7.1/+10.4 | **~54W** | **~80W** |
| **最大正弦（满功率）** | 10W | 10×16/0.9=178W | 10×24/0.9=267W | 2.6 | +27/+40 | **~208W** | **~310W** |

> **注**：最大正弦工况持续时间极短，实际音频内容 PAPR（峰均比）约 10dB，真实连续功耗远低于最大值。  
> 整机电源建议按**典型 ×3 安全系数**配置：16ch 推荐 **200W AC-DC**，24ch 推荐 **350W AC-DC**。

### D.2 热阻计算与结温验证

**ADSP-21569（DSP）**：
```
ADSP-21569 数据手册: Tj_max = 85°C, Rjc ≈ 3.5°C/W (BGA 封装估算)
典型环境温度 Ta = 45°C (密闭腔体内部，参考音柱安装环境)
导热垫 Rcs ≈ 0.5°C/W，小铝散热片 Rsa ≈ 8°C/W

Tj = 45 + 2.6 × (3.5 + 0.5 + 8) = 45 + 31.2 = 76.2°C
降额要求 Tj ≤ 85 × 0.8 = 68°C  ← 76.2°C > 68°C，⚠️ 需优化！
对策：将导热片规格升级至 Rsa ≈ 5°C/W 散热片，或适当散热孔通风
优化后: Tj = 45 + 2.6 × (3.5 + 0.5 + 5) = 45 + 23.4 = 68.4°C ≈ 临界满足
建议留额外余量：若可开导热孔 Ta_腔体 降至 40°C，则 Tj ≈ 63.4°C < 68°C ✓
```

**TAS5825M（功放）**：
```
TAS5825M: Tj_max = 150°C, Rjc = 2.7°C/W（VQFN 封装，有暴露焊盘）
每片功耗 2.2W（2ch × 1.1W），PCB 铜皮散热（无外置散热片，依赖 PCB 铜面积）
Rcs(PCB焊盘铜皮) ≈ 5°C/W，Rsa(PCB对流) ≈ 15°C/W

Tj = 45 + 2.2 × (2.7 + 5 + 15) = 45 + 50 = 95°C
降额要求 Tj ≤ 150 × 0.8 = 120°C  → 95°C < 120°C ✓ 满足（PCB 散热即可）
```

### D.3 密闭音柱腔体散热注意事项

1. **热分区设计**：功放散热区域与扬声器腔体隔开，避免热量积聚在扬声器单元背腔。
2. **PCB 布局**：TAS5825M 背面散热焊盘（exposed pad）必须与 PCB 铜皮充分焊接，铜皮面积 ≥ 300mm²/片，打散热过孔矩阵（via stitching）将热量导向内层铜。
3. **DSP 散热**：ADSP-21569 BGA 背面无 exposed pad，散热依赖顶面铜皮 + 导热硅脂 + 铝散热片贴合（约 20×20×5mm 铝散热片）。
4. **被动散热孔**：密闭音柱腔体顶部/底部建议开百叶窗散热孔（IP20 等级），保证自然对流，有效降低腔内 Ta 5–8°C。
5. **热保护**：TAS5825M 内置热保护（160°C 关断），DSP 通过软件监控核温（ADC 内部温度传感器），超过 75°C 触发降功率告警。
6. **电源模块**：明纬 MSP-450-24 本体自带散热风扇（密闭机型选明纬 MeanWell MDR 或 NPF 无风扇版本），若用内置电源需独立散热区。

## E. 16ch vs 24ch 接口影响汇总

| 维度 | 16ch 方案 | 24ch 方案 | 影响等级 |
|------|----------|----------|---------|
| Codec | 1× ADAU1966A | 3× ADAU1962A（或 2× ADAU1966A） | 中（PCB 面积+） |
| TDM 路数 | 1 路（SPORT0） | 2 路（SPORT0+1） | 低（SHARC 有余量）|
| 功放片数 | 8× TAS5825M | 12× TAS5825M | 中（PCB 面积+） |
| 功放总热耗 | 17.8W（典型语音）| 26.7W（典型语音）| 高（散热方案差异）|
| 整机 BOM 成本 | 见 bom_v0.md | 见 bom_v0.md | 见 BOM |
| 电源容量 | 200W 足够 | 350W 建议 | 中（AC-DC 可共用 MSP-450-24）|
| PCB 尺寸 | ~160×120mm 可行 | ~200×160mm（估算）| 中（需结构确认）|
