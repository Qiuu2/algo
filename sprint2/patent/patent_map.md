## A. 三篇核心论文公式提炼

### A.1 Dolph 1946 — 最关键公式

**论文**：C. L. Dolph, "A Current Distribution for Broadside Arrays Which Optimizes the Relationship Between Beam Width and Side-Lobe Level," *Proc. IRE*, vol. 34, pp. 335–348, June 1946.

#### 公式 1：Chebyshev 参数 z₀（旁瓣电平决定波束宽度）

```
T_M(z_0) = r,    z_0 ≥ cos(π/(2M))

M = 2N-1（2N 阵元）或 M = 2N（2N+1 阵元）
r = 主瓣/旁瓣幅度比（如 -26dB → r = 10^{26/20} ≈ 20）
```

**物理含义**：给定旁瓣电平，唯一确定参数 z₀，进而决定最优加权系数。这是 Dolph-Chebyshev 方法的核心入口公式。旁瓣越低（r 越大），z₀ 越大，主瓣越宽——这就是波束宽度–旁瓣的最优折中关系。

**在本项目中的用途**：设计 FAS 系数时，输入目标旁瓣电平（建议 -26 dB），用此公式求 z₀，再递推求 16 阵元的 Dolph-Chebyshev 权重向量，离线存入 ADSP-21569 SRAM。

#### 公式 2：方向图完整解析式

```
E(θ) = cos{ M · arccos[ z_0 · cos(πd/λ · sinθ) ] }
```

**物理含义**：最终方向图是一个 arccos 嵌套的解析表达式，可直接用于快速仿真评估任意角度的辐射电平。主瓣宽度、旁瓣位置、旁瓣高度均由 z₀ 和 d/λ 唯一确定。

### A.2 Boone et al. 2009 — 最关键公式

**论文**：M. M. Boone, W.-H. Cho, J.-G. Ih, "Design of a Highly Directional Endfire Loudspeaker Array," *J. Audio Eng. Soc.*, vol. 57, no. 5, pp. 309–325, May 2009.

#### 公式 1：有稳定因子的最优滤波器

```
F_opt^T(ω) = W^H(ω) · [S_zz + β(ω)·I]^{-1}
              ─────────────────────────────────
              W^H(ω) · [S_zz + β(ω)·I]^{-1} · W(ω)

S_mn = sin[k(z_m - z_n)] / [k(z_m - z_n)]（端射阵相干矩阵）
```

**物理含义**：β 是稳定因子（regularization）。β=0 时为最大指向性解，β 增大时退化为 DAS。低频区 β 需增大以避免驱动增益爆炸（功率指数 PI 过高）。

**在本项目中的用途**：子带 FAS 实现的理论基础。每个子带独立计算 F_opt，低频子带用大 β，高频子带用小 β，在 Python 中用 numpy 求逆实现，结果写入 ADSP-21569 子带系数表。

#### 公式 2：最大指向性指数上界

```
DI_max = 20 · log10(N)  [dB]（端射阵）
```

**物理含义**：N 阵元端射阵列的理论上限。16 元 = 24.1 dB，24 元 = 27.6 dB。实际受稳定因子约束，低频可达值约 12~16 dB。

### A.3 Zhang et al. 2024 — 最关键公式

**论文（已核实真实信息）**：  
Yankai Zhang, Qian Xiang, Qiaoxi Zhu,  
"Design of Differential Loudspeaker Line Array for Steerable Frequency-Invariant Beamforming,"  
*Sensors*, vol. 24, no. 19, article 6277, 2024. DOI: 10.3390/s24196277.  
作者机构：安徽师范大学 + 悉尼大学。发表日期：2024年9月27日。

> **引用更正**：任务书中称此文为"疑为 Zhang 2024"。经页面确认，真实标题、作者、期刊、年份均已核实如上，非"差分线阵"而是"可控频率不变差分波束成形"，略有侧重差异，以上信息为准。

#### 公式 1：目标差分方向图（N 阶多项式）

```
B^{(N)}(θ) = Σ_{n=0}^{N} α_{N,n} · cos^n θ
```

**物理含义**：N 阶差分阵的理想方向图是 cos θ 的 N 阶多项式。当阵元间距 δ << λ 时，此方向图近似频率不变（不随频率改变形状）。系数 α 由转向角 θ_s 和旁瓣约束确定。

**在本项目中的用途**：为"子带差分+相控混合"差异化方向提供理论基础。低频子带（<1kHz）用差分方向图（紧致孔径，频率不变），高频子带（>1kHz）用 Dolph-Chebyshev 加权 FAS，两段平滑拼接。

#### 公式 2：多约束优化（WNG + MSE 折中）

```
min_{α_N}  |B^{(N)}(Θ_SL)|^2   s.t.  A^T · α_N = b

Θ_SL ∈ [0°, θ_s - Δ/2] ∪ [θ_s + Δ/2, 180°]
```

**物理含义**：在主瓣转向约束下最小化旁瓣能量，凸优化（CVXPY/CVX 求解）。额外引入 WNG 下界约束可确保系统鲁棒性（限制低频驱动增益）。

## D. 差异化技术方向推荐（3–5 个）

以下方向综合考虑 ADSP-21569 算力裕量（约 1 GHz，双核 SHARC+）、专利规避需求、以及可申请性。

---

### 方向 1：自适应频率不变波束（子带 Dolph-Chebyshev + 频段拼接）

**技术描述**：将工作频段（500 Hz – 4 kHz）划分为 8–16 个子带，每个子带独立计算 Dolph-Chebyshev 最优加权，使得各子带 -6dB 波束宽度保持一致（如 30°）。在 ADSP-21569 中采用多通道 FIR 子带滤波器组（polyphase 结构），实现计算高效的频率不变指向性。

**差异化点**：现有专利多针对宽带单一滤波器方案，本方案创新点在于"基于等波束宽度约束的子带 Dolph-Chebyshev 系数自适应优化"，结合 SHARC 的子带定点实现策略（精度误差控制在 1e-3 以内）。

**专利可申请性**：子带分析合成框架为公知，但"等 -6dB 波束宽度约束 + 子带 Chebyshev 优化 + SHARC 定点实现"的完整系统组合有一定新颖性，可考虑申请发明专利（国内），重点保护系数计算方法和 DSP 实现结构。

**算力评估**：16 阵元、8 子带，每子带 16 路 FIR，ADSP-21569 单核算力充裕（FAS 主体约 15 MIPS，远低于 1 GHz 上限）。

---

### 方向 2：子带差分+相控阵混合（低频差分 / 高频相控）

**技术描述**：1 kHz 以下采用 N 阶差分波束成形（小孔径高指向），1 kHz 以上采用 Dolph-Chebyshev 加权 FAS，两段在频域平滑拼接。差分阶数随频率自适应调整（低频用 3–4 阶，中高频退化为 DAS 叠加 Chebyshev 加权）。

**差异化点**：Zhang 2024 论文提出差分扬声器线阵理论，但其实现面向 broadside，本方案将差分方向图与 Dolph-Chebyshev 相控阵在子带层面无缝混合，解决"可闻声线阵低频指向性差"的核心痛点（CTO 已指出 ≥1kHz 才有定向需求，本方案正好将差分阵优势用于补偿 500–1000 Hz 过渡段）。

**专利可申请性**：差分阵用于扬声器（非麦克风）且与相控阵混合的方案文献极少，可申请性较强。建议重点保护：混合策略（子带划分准则 + 两段系数融合方法）。

**算力评估**：差分阵系数计算离线完成，实时开销与方向 1 相当，完全在 ADSP-21569 算力范围内。

---

### 方向 3：基于扬声器自校准的在线波束成形修正

**技术描述**：利用内置或外置测量话筒，在部署环境中对各单元的幅频/相频特性进行测量，自动修正单元一致性误差（±3dB 容差范围内的偏差），更新 FAS 系数，消除生产/老化误差对波束形状的影响。ADSP-21569 ARM 核运行校准程序，SHARC 核实时处理信号，两核协同。

**差异化点**：现有产品极少将自校准集成到音柱 DSP 中。苹果近期有相关专利（US20220053281），但针对可穿戴设备。博物馆/广播场景的固定安装音柱 + 环境自适应校准是空白市场，且利用 SC589/21569 的双核异构架构实现本身是新颖组合。

**专利可申请性**：可申请性强，差异化点明确（应用场景 + 实现架构）。建议与校准算法（最小二乘拟合 + 有限次测量）一起保护。

---

### 方向 4：近场动态聚焦分区广播（博物馆/商场场景）

**技术描述**：在近场（1–10 m）对不同距离/角度的区域实施独立聚焦，同时向 A 区播放中文讲解、向 B 区播放英文讲解，两区声压交叉污染 ≤ -15 dB。利用近场阵列因子修正（不能用远场近似），在 ADSP-21569 中实现近场相位补偿矩阵计算。

**差异化点**：现有线阵产品（含竞品）基本只实现远场定向，近场多区独立聚焦在中端产品中极为罕见。US9119012（2015 年授权约 2035 到期）覆盖"个人音频焦点"场景，但其权利要求偏向小型设备，博物馆固定安装大功率多区广播有差异化空间。

**专利可申请性**：可申请"定向音柱近场多区独立广播系统及方法"，保护近场相位补偿方法 + 多区信号路由架构。风险提示：需请代理人比对 US9119012 权利要求，确认不侵权。

**算力评估**：近场修正矩阵较大（L×M 维度），需预计算并存片内 SRAM，实时卷积不增加额外计算。可行。

### 方向 5：基于 AI 辅助的声场校准与方向图优化

**技术描述**：离线阶段，用轻量级神经网络（MLP 或简单 CNN）学习房间反射模型，预测不同房间环境下的最优 FAS 系数；在线阶段，ADSP-21569 ARM 核调用预训练模型推理（INT8 量化，约 0.5–2 MOPS），实时微调波束权重以补偿房间声学变化。

**差异化点**：AI 辅助声场优化在高端会议系统（Shure/Biamp/Cisco）有所涉及，但均针对麦克风阵列，未见用于扬声器线阵波束成形的公开专利。ADSP-21569 ARM 核的计算能力（600 MHz Cortex-A5）足以运行轻量推理，这一组合（SHARC DSP 波束成形 + ARM 核 AI 校准）具有明确新颖性。

**专利可申请性**：较强，可申请"基于神经网络的定向音柱自适应波束成形方法"。但注意 AI 方法类专利需要具体技术特征支撑（不能过于抽象），建议配合具体网络架构和训练数据描述申请。

**算力评估**：INT8 轻量 MLP 推理（输入：房间特征向量，输出：系数微调量）在 ARM 核运行，估计 <10 MOPS，完全可行。

## E. 最高专利风险点总结

**最高工程风险**：  
**EP1986464A1（端射扬声器阵列，Boone/Delft）**  
若该专利在中国有同族授权专利且仍有效，则 ITC 采用端射 + 稳定因子最优波束成形的核心方案存在侵权风险。这是唯一一项与 ITC 技术方案高度重叠且来源于 ITC 参考文献的专利，必须优先由专利代理人核查。

**次高工程风险**：  
**US8135158B2（Fincham 线阵综合专利）**  
权利要求范围可能较宽，约 2027 年到期。若中国无同族则中国市场无风险；若有则需比对。

**整体评估**：  
本项目技术路线（相控阵可闻声线阵 + Dolph-Chebyshev 加权 + ADSP-21569 FAS）在核心数学原理上属于公有领域，专利风险主要来自具体实现方案的专利覆盖。通过选择差分+相控混合路线（方向 2）或频率不变子带方案（方向 1），可有效规避 Boone 端射最优波束成形专利的潜在风险。
