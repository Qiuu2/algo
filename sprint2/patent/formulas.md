## 1. Dolph 1946 — 阵列方向图与 Chebyshev 最优加权

**论文信息**：  
C. L. Dolph, "A Current Distribution for Broadside Arrays Which Optimizes the Relationship Between Beam Width and Side-Lobe Level," *Proc. IRE and Waves and Electrons*, vol. 34, pp. 335–348, June 1946.

---

### 1.1 阵列场方向图（2N 阵元，偶数阵列）

```
|E_2N(θ)| = | Σ_{k=0}^{N-1} I_k · cos{ (2k+1)/2 · (2πd/λ)·sinθ } |     … (1)
```

**物理含义**：N 对关于中心对称的阵元（总计 2N 个），各阵元加权电流为 I_k，d 为间距，λ 为波长，θ 为与阵面法线的夹角。场强是各路相干叠加的模值。

**项目用途**：本项目 16 阵元对应 N=8，用此公式直接计算 broadside（侧射）方向图；配合下文 Chebyshev 映射求出最优 I_k。

---

### 1.2 空间频率变量替换

```
u = (πd/λ) · sinθ                                                           … (3)
```

**物理含义**：将物理角度 θ 映射到无量纲"空间频率" u。d/λ 决定可见区范围（d=λ/2 时 u ∈ [0, π/2]；d=λ 时 u ∈ [0, π]）。

**项目用途**（算例几何已更新 **d=55mm**[L1拆机实测]，DEC-S3-GEOM-01；旧 d=30mm 已作废）：500 Hz 时 d/λ ≈ 0.080，4000 Hz 时 d/λ ≈ 0.641，8000 Hz 时 d/λ ≈ 1.282；仿真时用此公式将角度扫描转为 u 域计算，判断栅瓣出现条件（d/λ ≥ 1 时栅瓣入侵可见区 → **6.2–8kHz 段有栅瓣**[L1/几何]）。

---

### 1.3 方向图多项式化（2N 阵元）

```
F_2N(u) = | Σ_{k=0}^{N-1} I_k · cos(2k+1)u |                               … (4)

令 x = cos u，则 F_2N → G_2N(x) = Σ_{q=0}^{N} Σ_{k=q}^{N} I_k · A_{2q}^{2k} · x^{2q}  … (13)
```

**物理含义**：方向图可以写成 x = cos u 的多项式，便于用 Chebyshev 代数工具操纵零点位置。

---

### 1.4 Chebyshev 多项式定义

```
T_n(z) = cos(n · arccos z),   -1 ≤ z ≤ 1                                   … (15)
```

**物理含义**：T_n(z) 在 [-1, 1] 内以等幅振荡 n-1 次达到 ±1，在 |z| > 1 单调递增/递减。正是这一"等幅旁瓣"特性使 Chebyshev 加权成为最优解。

---

### 1.5 z₀ 参数与主/旁瓣比

**指定旁瓣电平比 1/r 时，z₀ 满足**：

```
T_M(z_0) = r,    z_0 ≥ cos(π/(2M))                                         … (24)

其中 M = 2N-1（2N 阵元）或 M = 2N（2N+1 阵元）
```

**物理含义**：z₀ 是 Chebyshev 多项式的"压缩参数"，将方向图可见区 x ∈ [-1, 1] 映射到 Chebyshev 高振荡区 z ∈ [-z₀, z₀]。r 是主瓣/旁瓣幅度比（如旁瓣 -26 dB 则 r ≈ 20）。

**项目用途（核心公式）**：给定目标旁瓣电平（如 -26 dB），用此公式反解 z₀，然后通过方程 (22)/(23) 迭代求解全部加权系数 I_q。这是 ADSP-21569 上 FAS 系数计算的入口。

---

### 1.6 最优阵列方向图完整解析式

```
E(θ) = cos{ M · arccos[ z_0 · cos(πd/λ · sinθ) ] }                         … (27变体)
```

**物理含义**：最终方向图是一个余弦嵌套的解析表达式，其中 z₀ 控制旁瓣，d/λ 控制主瓣宽度，M（阵元数相关）控制零点数量。

---

### 1.7 加权系数递推公式（van der Mass / Ward 简化版）

```
W_n(i) = (n-1)/(n-i) · Σ_{s=0}^{M} C(i-2, s) · C(n-i, s+1) · α^{s+1}

其中 M = i-2 if i ≤ (n+1)/2,  M = n-i-1 if i ≥ (n+1)/2
     α = 2·[log(S + sqrt(S²-1))]/(n-1)，S = 10^{SLL/20}
     边界条件：W_n(1) = W_n(n) = 1
```

（来源：H. R. Ward, *Properties of Dolph-Chebyshev Weighting Functions*, IEEE Trans. AES, Sep 1973，即补充参考文献）

**物理含义**：闭式递推，只需输入阵元数 n 和旁瓣电平 SLL（dB），即可逐一算出全部归一化权重 W_n(i)。不需要矩阵求逆，适合 DSP 预计算。

**项目用途**：在 ADSP-21569 初始化阶段（离线）计算 16/24 阵元 Dolph-Chebyshev 权重表，存入片内 SRAM，运行时 FAS 直接查表相乘。

---

### 1.8 波束宽度–旁瓣折中关系（归一化带宽，Ward 1973）

```
TB_3  = (L · sinθ_3) / λ      ← -3 dB 波束宽度（归一化）
TB_6  = (L · sinθ_6) / λ      ← -6 dB 波束宽度（归一化）
TB_SLL= (L · sinθ_SLL) / λ    ← 旁瓣零点位置（归一化）

其中 L = (N-1)·d 为阵列孔径
```

**物理含义**：归一化波束宽度 TB 随旁瓣电平 SLL 单调增大（旁瓣压越低，主瓣越宽）。参见 Ward 1973 Fig. 3 曲线。

**项目用途**（算例几何已更新 **d=55mm**）：快速评估"16 阵元 × 55mm × 2kHz"能否满足 -6dB 波束宽度 ≤ 30° 的设计目标。孔径 L = 15×55mm = **825mm**，在 2kHz 时 L/λ ≈ 4.81，代入 Ward 曲线查 TB_6 即得（d=55 仿真 BW@2k=14.5°，余量大）。

## 2. Boone et al. 2009 — 端射阵列高指向性扬声器

**论文信息**：  
M. M. Boone, W.-H. Cho, and J.-G. Ih, "Design of a Highly Directional Endfire Loudspeaker Array," *J. Audio Eng. Soc.*, vol. 57, no. 5, pp. 309–325, May 2009.

---

### 2.1 端射阵列传播向量（核心差异）

```
W(θ, φ, ω) = [ Γ_1·exp(jkz_1 cosθ),  Γ_2·exp(jkz_2 cosθ), ..., Γ_N·exp(jkz_N cosθ) ]^T
                                                                                … (3)
```

**物理含义**：端射（endfire）配置下，阵元沿 z 轴排布（z_i 为第 i 阵元纵向位置），传播因子为 exp(jkz_i cosθ)，与 broadside 的 exp(jkx_i sinθ) 形式不同。方向图最大值在 θ=0°（轴向方向）。

**项目用途**：本项目定向音柱为端射布局，用此矩阵构建系统方向图计算，而非 Dolph 的横向阵公式。

---

### 2.2 指向性因子（Directivity Factor）

```
Q(ω) = max_{θ,φ}{ F^H(ω) · W*(θ,φ,ω) · W^H(θ,φ,ω) · F(ω) }
        ─────────────────────────────────────────────────────
                    F^H(ω) · S_zz^T(ω) · F(ω)               … (4)
```

其中相干矩阵元素：

```
S_mn(ω) = (1/4π) ∫∫ W_m(θ,φ,ω) · W_n*(θ,φ,ω) · sinθ dθ dφ … (5)
```

对端射阵列，相干矩阵元素可化简为 sinc 函数：

```
S_mn = sin[k(z_m - z_n)] / [k(z_m - z_n)]                    … (6)
```

**物理含义**：指向性因子 Q 是目标方向声强与全向辐射功率之比，用矩阵形式统一表达滤波器向量 F(ω) 对方向性的贡献。

**项目用途**：DI（dB）= 10 log Q 是定向音柱最核心的性能指标，用此公式在 Python 仿真中评估不同 FAS 权重下各频率点的 DI。

---

### 2.3 噪声灵敏度（白噪声增益 / 功率指数）

```
Ψ(ω) = F^H(ω) · F(ω) / [F^H(ω) · W*(ω) · W^T(ω) · F(ω)]   … (7)
```

**物理含义**：Ψ 衡量滤波器对白噪声（通道不一致误差）的放大倍数。对于扬声器阵列，Ψ 直接关联驱动功率效率（Boone 称为 Power Index，PI）。Ψ 过大意味着低频需要极大驱动增益，实现困难。

**项目用途**：最优波束成形在低频 Ψ 急剧增大，需通过稳定因子 β 加以约束，权衡 DI 与 PI。

---

### 2.4 最优波束成形解（有稳定因子 β）

```
F_opt^T(ω) = W^H(ω) · [S_zz + β(ω)·I]^{-1}
              ─────────────────────────────────
              W^H(ω) · [S_zz + β(ω)·I]^{-1} · W(ω)          … (16)
```

**物理含义**：β 是"稳定因子"（regularization），β→0 趋近纯最优解（最大 DI），β 增大则趋近 DAS（Delay-and-Sum）。β 控制 DI 与 PI 之间的折中。

**项目用途**：β(ω) 随频率可变，低频段适当加大 β 以限制驱动增益，高频段减小 β 充分利用指向性。此公式可直接在 Python 中用 numpy 求逆实现，结果写入 ADSP-21569 的子带系数表。

---

### 2.5 最大指向性指数（阵元数决定上界）

```
DI_max = 20 · log10(N)   [dB]                                  … (20)
```

**物理含义**：端射 N 阵元阵列，在最优激励下可达理论最大 DI，比等效 broadside 阵高 3dB。

**项目用途**：16 阵元理论 DI_max = 24 dB，24 阵元 = 27.6 dB。实际在中低频受 β 约束，可达值约为 12~18 dB。这是向 CTO 汇报 DI 指标上界的依据。

---

### 2.6 高频归一化截止频率

```
f_h = c / (2d)                                                  … (17)
```

**物理含义**：半波长（d=λ/2）无栅瓣的归一化频率上限。**d=55mm**[L1拆机实测]，c=343 m/s，f_h = 343/(2×0.055) ≈ **3118 Hz**（半波长判据）；broadside 严格判据 d=λ → **6236 Hz**。

**项目用途**（算例几何已更新 **d=55mm**）：500Hz–5kHz 安全，**6.2–8kHz 出现栅瓣**[L1/几何]（6.24kHz 起栅瓣峰 0dB、泄漏 78–89%[L2 仿真]）。应对：强指向规格上限由 8kHz 降级为 6kHz(对内)/5kHz(对外保守)（方向1，CTO 采纳）。

## 3. Zhang et al. 2024 — 差分线阵可控频率不变波束成形

**论文信息（已核实）**：  
Yankai Zhang, Qian Xiang, Qiaoxi Zhu, "Design of Differential Loudspeaker Line Array for Steerable Frequency-Invariant Beamforming," *Sensors*, vol. 24, no. 19, article 6277, 2024. DOI: 10.3390/s24196277.  
（注：原文件名 "Design_of_Differential_Loudspe.pdf" 经页面确认，真实标题为以上，非"Zhang 2024"原始推测，作者为安徽师范大学 + 悉尼大学，发表于 MDPI Sensors 2024年9月。）

---

### 3.1 远场辐射方向图（线阵基本公式）

```
B(k, θ) = Σ_{l=1}^{L} w_l*(k) · e^{-ikx_l cosθ}             … (2)

向量形式：B(k, θ) = w^H(k) · g(k, θ)

其中 g(k, θ) = [e^{-ikx_1 cosθ}, ..., e^{-ikx_L cosθ}]^T     … (5)
     k = 2πf/c（波数），x_l = -(L+1)δ/2 + lδ（第 l 阵元位置）
```

**物理含义**：远场声压（归一化）等于加权向量 w 与阵元导向向量 g 的内积。θ 为相对 x 轴正方向（线阵轴）的夹角，broadside 方向为 θ=90°。

**项目用途**：仿真中计算任意频率、任意角度的归一化方向图，是差分波束成形的出发点。

---

### 3.2 白噪声增益（WNG）与指向性因子（DF）

```
WNG(k) = |B(k, θ_s)|^2 / [w^H(k) · w(k)]                     … (6)

DF(k) = π|B(k, θ_s)|^2 / ∫_0^π |B(k,θ)|^2 dθ
       = |B(k, θ_s)|^2 / [w^H(k) · Γ(k) · w(k)]              … (7)

其中 Γ(k) = (1/π) ∫_0^π g(k,θ) · g^H(k,θ) dθ
     [Γ]_{m,n}(k) = J_0[k(i-j)δ]                              … (8)
```

**物理含义**：WNG 衡量波束成形对通道噪声的鲁棒性（对扬声器阵列即辐射效率），DF 衡量空间定向能力。两者在差分阵设计中存在本质折中：差分阵 DF 高但 WNG 低（低频驱动增益巨大）。

**项目用途**：WNG 是本项目实现低频覆盖（500 Hz）的最大制约。通过约束 WNG ≥ WNG_min 可避免过大驱动电平损坏单元。

---

### 3.3 目标差分方向图（N 阶差分阵理想模式）

```
B^{(N)}(θ) = Σ_{n=0}^{N} α_{N,n} · cos^n θ                   … (9)

向量形式：B^{(N)}(θ) = α_N^T · c_N,  c_N = [1, cosθ, ..., cos^N θ]^T  … (13)
```

**物理含义**：N 阶差分阵的理想方向图是 cosθ 的 N 阶多项式，系数 α_{N,n} 决定方向图形状。当阵元间距远小于波长时，差分阵实现近似频率不变方向图。

**项目用途**：用于快速计算目标方向图的系数向量 α_N，不依赖具体频率，仅由阶数 N 和转向角 θ_s 决定。

---

### 3.4 转向约束与系数求解

```
主瓣增益归一化：B^{(N)}(θ_s) = 1
主瓣方向梯度为零：∂B^{(N)}(θ_s)/∂θ_s = 0

矩阵形式：A^T · α_N = b

A = [1   cosθ_s   ...   cos^N θ_s  ]^T
    [0   sinθ_s   ...   N·cos^{N-1}θ_s · sinθ_s]^T
b = [1, 0]^T                                                    … (10-12)
```

**物理含义**：两个约束确保目标方向主瓣增益为 1 且方向图在该处极大，由此确定系数向量 α_N，实现波束可控转向。

---

### 3.5 多约束最优化（MSE 最小化 + WNG 约束）

```
min_{α_N}  |B^{(N)}(Θ_SL)|^2       旁瓣区能量最小化
s.t.        A^T · α_N = b            主瓣转向约束

旁瓣区定义：Θ_SL ∈ [0°, θ_s - Δ/2] ∪ [θ_s + Δ/2, 180°]      … (17-18)
主瓣宽度约束：Δ ≤ 2·min(θ_s, 180°-θ_s)                        … (19)
```

**物理含义**：在满足主瓣指向和增益约束的前提下，最小化旁瓣区辐射能量，用 CVX 或 CVXPY 求解凸优化，得到最优 α_N。

**项目用途**：本文方法直接给出可在 ADSP-21569 实现的方向图设计流程。在 Python 中用 CVXPY 求解后生成系数表，写入 DSP。子带实现时每个子带独立求解，实现频率不变波束。

---

### 3.6 Jacobi-Anger 展开（模式域 → 阵元权重）

```
e^{-ikx_l cosθ} = Σ_{n=0}^{∞} (-ikx_l)^n / n! · cos^n θ      (Jacobi-Anger 展开近似)
```

**物理含义**：将阵列方向图从物理域（阵元权重）转换到模式域（cos^n θ 系数），使得 α_N 直接对应物理加权向量 w(k)，从而将目标差分方向图映射到实际阵元激励信号。

**项目用途**：这是差分波束成形"从目标方向图反推阵元权重"的关键桥梁，实现可控频率不变波束的工程计算路径。
