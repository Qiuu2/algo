# 定向音柱算法有效性 — 整机验证测试文档

> 目的：在**真实硬件链**（AD-EXKIT 跑 M2 → DAC → 功放 ×4(8路) → 16 元阵列）上，
> 系统性确认「M2 FIRA broadside 波束算法**真的产生了定向效果**」，并量化到可与仿真对账的程度。
> 这是阶段④ 软件收口后、阶段⑤ 自研 PCB 之前的**整机算法验证**——也是第一次用真功放+真阵列听到算法效果。
> 维护：PM lead。权威数据源：M2 board PASS（DEC-S6-M2-BOARD-PASS-01）+ Dolph 权重表 + 阵列因子仿真。

---

## 0. 先读：这份测试「证明什么 / 不证明什么」（L 分级与边界）

**证明（本测试的目标）**：
- 整条电-声链通（8 路都出声、无哑路/异响）；
- 8 路幅度锥化 = Dolph 设计（接线顺序/极性对）；
- **正前方有定向波束、侧面明显衰减**——即算法在真实声学上有效；
- 实测指向性与 [L2] 阵列因子仿真大致对账（量级一致）。

**不证明（诚实边界，别越界宣称）**：
- **不是** R3 消声室 L1 终验：本测试在普通房间，有反射/驻波，是 **[L1 现场实测 / 普通房间口径]**，半定量；绝对 SPL、精确波束宽度、≥90dB 对外承诺仍待 R3。
- **不是** bit-exact 数值正确性：逐子带 CRC（M2_Q_BOUNDARY_SURVEY §3）是另一道独立门，本测试不替代。
- 当前是 **broadside（正前固定波束）**，**不是**波束偏转，也**不是**多档扩散角（那些是后续立项）。

**关键数字来源标级**：
- Dolph 权重 / 各路相对电平 = **[L2 设计权重 / 确定性比值派生]**（g_dolph_w8_q15 桌面 Q15 量化值，header 自标 L2/dual-track；板上是「应用」该权重非「实测」出它；相对电平=纯 20log10 比值，不依赖来源等级）；
- 波束宽度 / 旁瓣 / 栅瓣 = **[L2 阵列因子仿真 + L1 几何]**（理想点源/自由场；与档案 AC-WP01 + 本项目 MATLAB 双轨一致）；
- M2 板上读数 = **[L1/EZKIT]**（DEC-S6-M2-BOARD-PASS-01）。

## 2. 测试项（按顺序，逐项 PASS/FAIL）

### 测试 0 — DSP 端在出正确波束（电域，不出声也能确认）
板上 Expressions 读，跟 1B PASS 一致即 PASS：

| 变量 | PASS 判据 |
|---|---|
| `g_m2_fira_inloop` | = 1 |
| `g_m2_setup_rc` / `g_m2_valid` / `g_m2_fg_beam_live` | 0 / 1 / **1** |
| `g_m2_out_nonzero` | 持续增长 |
| `g_m1_rx_block_count` ≈ `g_m2_poll_count` | 都增长且相等 |
| `g_m2_overrun_count` | **≈0（主判据：证波束每帧塞得下）** |
| `g_m2_beam_cyc_max` | < 1,333,333（帧周期；**板测基线≈830903≈62%墙钟，余量仅1.6×非「远低于」**；显著超~110万或 overrun 涨=异常。墙钟口径含等FIR忙等，勿与纯核~130µs[L2]比，R60纪律）|

→ 全绿 = DSP 在出正确的 8 路 Dolph 波束信号。**先过此项，再上功放。**

### 测试 1 — 接线顺序与单元健康（**逐路 solo 法，2026-06-29 修订**）

> **⚠ [2026-07-08 极性口径订正 → deliverables/algorithm_validation/BEAM_POLARITY_CLOSURE_20260708.md]**：本节逐路 solo **等幅 SPL 左右对称**只能判**接线顺序/无死路**,**判不了极性**(极性反=波形取反,幅度不变→SPL 一样)。原「极性确认对」措辞**撤回**;极性须**成对声学相消**或**电池纸盆**法(见闭合文档)。07-08 在竞品喇叭 rig 上即用成对法查出 **C2/C4 两对反极性**、翻正后 M2 波束由平恢复为有指向。

> **⚠ 方法订正（DEC-S6-TEST1-METHOD-01，2026-06-29 现场实证，详见 deliverables/algorithm_validation/TEST1_WIRING_FINGERPRINT_FIELDLOG_20260629.md）**
> 原方法「全阵齐放同一 2kHz 单音 + 手机分贝计贴近测每对」**已退役——测不准，不要用**。
> 根因 [L3 解析]：8 对同时放同一相干单音时，2kHz 半波长仅 85.8mm（元件间距 55mm=0.32λ），全阵在近场（远场 2L²/λ≈7.9m，1m 及更近全在近场）形成相干干涉场；贴近任一点读到的是 16 元叠加后该点的局部峰/谷（挪几 cm 差 10dB），**不是那一对的电平**。Dolph 锥化全程仅 6dB，被 ±10dB 干涉抖动盖掉，信噪比<1，指纹无法提取。2026-06-29 首测即坐实：c5 读 −9.7dB（落进相消零点），逐路 solo 复测同一 c5 反而是最响档之一（111.6/111.4dB），证明那是干涉零点非弱对。
> 另注：对称串联对几何（c0={元件0,15} 劈在阵列两端相距 825mm；c7={7,8} 居中相距 55mm）使「一个麦位=一对」不成立——贴近外侧对只测到一个端头元件，中心对两元件都贴着麦 → 即便等幅驱动也造出"中心高外侧低"假锥化，与接线对错、与 Dolph 权重均无关。

**正确方法 = 逐路 solo（一次只让一路出声），手机分贝计固定贴近距离逐个量。**

**1A 接线顺序 + 单元健康（必做，等幅外部驱动即可，不依赖算法）**
逐路 solo，麦固定贴近各**单元**测 SPL。等幅驱动下应**大致齐平 + 左右镜像对称**：
- **PASS①（接线/极性对）**：每路两镜像单元 {c,15-c} SPL 差 ≤ ~1.5dB（左右对称）；
- **PASS②（无哑路/弱单元）**：16 个单元无一比均值低 >6dB（>6dB=该单元开路/虚焊/反相，查串联对焊点）；
- 等幅驱动下**不呈 Dolph 锥化**（没过算法权重），整体 ~3dB 内起伏属正常单元公差，**不作判据**。
> 2026-06-29 实测 [L1 现场/室外空旷地/0.15V 2kHz 等幅]：16 单元 108.7~111.9dB（span 3.2dB），8 对左右差最大 0.9dB，零哑路 → **PASS，接线顺序与极性确认对**。

**1B Dolph 锥化验证（可选，仅当跑 M2 让算法加权时才测得到）**
M2 在跑波束，逐路 solo（只留一路输出、静音其余），麦**固定同一贴近距离**对各路近端单元测。设计锥化 [L2 设计权重/比值派生]：

| 阵列位置（对）| 该路 c | 相对中心电平 [L2派生] |
|---|---|---|
| **最中心对** | c=7 | **0 dB（最响）** |
| 往外 | c=6 | −0.26 |
| | c=5 | −0.79 |
| | c=4 | −1.59 |
| | c=3 | −2.69 |
| | c=2 | −4.13 |
| 次外对 | c=1 | **−5.95 dB（最小）** |
| **最外对** | c=0 | **−1.24 dB（抬边，比次外明显响 +4.7dB）** |

**只比 c0 vs c1**（两者都是"一个近端单元"，几何条件相同，比值≈ w0/w1 = +4.7dB，不受串联对几何干扰）：
- **c0 明显 > c1（+~4.7dB）** → Dolph 锥化已正确应用；
- c0 ≤ c1 → 查 c0(=28404) 权重是否真应用（Test 0 已绿、board PASS 时权重应已加载，可能性低）。
> 等幅外部驱动测不到 1B（无锥化）；1B 非整机有效性必要条件——综合判定 C 的防呆作用已由 1A（左右对称）+ 输出声音映射两独立证据满足，**1B 缺失不阻塞 Test 2/3**。

**总判据**：1A PASS（左右对称 + 无哑路）= 接线顺序/极性确认对，可上 Test 2/3。
**FAIL**：某对左右明显不对称 / 某单元 >6dB 偏低 → 按 §1 表查该路功放输入线序与串联对正负，修后重测。

### 测试 2 — 极性（broadside 正前最大）
放 **2kHz 单音**，麦/耳在阵列**正前方 1m、阵列中心高度**。

**PASS 判据**：正前方是**明显最大**（broadside 同相叠加）。
**FAIL**：正前方不是最大、或正前方有个**凹陷/抵消**（某处反相）→ **极性翻了**，查串联对接线 + 功放输入正负，逐对核对后重测。

### 测试 3 — 指向性 / 波束（★算法有效性的核心证据）

> **⚠ 方法订正（DEC-S6-TEST3-METHOD-01，2026-06-29 现场实证；待 critic + CTO，详见 deliverables/algorithm_validation/TEST1_WIRING_FINGERPRINT_FIELDLOG_20260629.md §8）**
> 原「1m + 2kHz 纯单音 + 手持各测一次」**测不出指向性，已订正**。两个硬伤 [L3 解析，python 复算]：
> 1. **1m 是近场，方向图没成形**：本阵 D=825mm@2kHz，**最后轴向极大值 N=D²/(4λ)≈0.99m**、远场 2D²/λ≈**7.9m**。1m 正落在近场菲涅尔边界——**轴向(0°)被菲涅尔分区压低，旁边反而更响**，故"0° 不是最大 / ±30~60° 比 0° 响"在 1m 是**预期假象，非算法坏**。1m 实测前后落差仅 ~1.7dB（要≥12），近场展不开。
> 2. **纯单音 + 室外风 + 手持 = 声场不稳定**：2kHz 干涉零点间距仅 85.8mm，室外风/湍流让干涉场漂移→音量随时间忽大忽小；手持晃几 cm 跨半波长→读数跳十几 dB。3 次样本同角度波动：±60° 达 16dB、±30° 8~11dB（陡坡区放大抖动），0°/±90° 因在方向图极值点(斜率≈0)反而稳。

**订正后的稳健测法（必须照此重测，否则数据无效）**：
1. **距离 ≥ 3~4m**（理想 ~8m）——脱离近场。室外空旷地可拉远；
2. **信号换带宽信号**：2kHz 中心 **1/3 倍频程粉噪** 或 **warble（±1/6 倍频程扫频）**，**不要纯单音**（带宽把梳状零点平均掉，读数立刻稳）；
3. **Leq 时间平均**：每角度用分贝计 **Leq 测 10~30s 等效值**，不读瞬时峰值；
4. **麦上三脚架固定**（不手持），地面标好各角度恒定半径弧线点；
5. **架高 ≥1.2~1.5m + 防风棉 + 挑无风时段**（清早），减地面反射与风扰；
6. **每角度扫 3~5 遍取均值±离散**；离散仍 >3dB = 场地太糙，换更空旷处，精确值终归 R3 消声室。
7. 复核 **0° 严格垂直阵面** + **在阵列长轴平面扫**（柱子躺平横扫，见测试1注记）。

放带宽信号，恒定半径绕**阵列中心**，各角度测 SPL（以 0° 为基准）：

| 角度 | 测什么 |
|---|---|
| 0°（正前）| 基准 SPL（Leq）|
| ±30° / ±60° / ±90°（正侧）| 相对 0° 落差（Leq）|

**PASS 判据（算法有效的硬指标，须在稳健测法下取得）**：
- **0° 最大**（远场+带宽下，近场假象消除）；
- **±90° 相对 0° 落差 ≥ 12 dB**（保守门；理论旁瓣 −20dB，留反射裕度）；
- 左右**大致对称**（±30°/±60°/±90° 左右两侧落差差 ≤ 6 dB）。

→ 满足 = **波束成立、定向有效、极性对称全对**，「算法在真实声学上有效」的实证。
**⚠ 算法不背锅**：M2 是 broadside LTI（稳定单音进→稳定出，Test0 overrun=0），时间抖动是声学/环境非 DSP；近场假象是物理非算法。**别据此改算法/build。**

### 测试 4 — 频率行为（印证物理，对账仿真）
分别放 **1k / 2k / 4k / 6k Hz** 单音，各测 0° 与 ±90°，并感受主瓣宽窄：

| 频率 | 仿真预期 -6dB 全角 [L2] | 现场预期感受 |
|---|---|---|
| 1kHz | 29.3° | 波束较宽 |
| 2kHz | 14.6° | 明显变窄 |
| 4kHz | 7.3° | 更窄 |
| 6kHz+ | 栅瓣（6.24kHz 起）| **侧面又冒声**（±50~90° 出现强瓣）|

**PASS 判据**：① 频率越高主瓣越窄（0° 越尖、侧面越低）；② **6kHz 以上侧面明显回升**（栅瓣，印证 d=55mm 上限）。
→ 这两条对上 = 实测物理与仿真一致，模型可信。

### 测试 5 —（可选，量化加分）SPL-vs-角度方向图
若有测量话筒：固定 1m，每 10° 测一圈（0°→±90°），画 SPL-vs-角度曲线，叠到本会话 [L2] 仿真方向图上。
**对账判据**：实测主瓣宽度与仿真同量级（普通房间允许 ±一档展宽），旁瓣被压低的趋势一致。

## 3. 综合判定 —「算法确认有效」的条件

全部满足即判 **算法整机有效 [L1 现场/普通房间口径]**：

| # | 条件 | 来自 |
|---|---|---|
| ✅ A | 测试 0 DSP 端全绿 | 测试 0 |
| ✅ B | 8 路都出声、无哑路/异响 | 测试 1 |
| ✅ C | 接线顺序/极性对（逐路 solo：左右镜像对称 + 无哑路；Dolph 锥化 1B 可选）| 测试 1（2026-06-29 修订法）|
| ✅ D | 正前方最大、无反相凹陷 | 测试 2 |
| ✅ E | **±90° 落差 ≥ 12dB + 左右对称** | 测试 3 |
| ✅ F | 高频变窄 + 6kHz 栅瓣回升 | 测试 4 |

**E 是核心**（定向的硬证据），C/D 保证 E 不是接错的巧合。
**任一 FAIL** → 按对应测试的 FAIL 分支修（顺序/极性/接线），不下结论。

**诚实声明**：判「有效」= **方向性在真实硬件上成立且与仿真量级一致**；**不等于**消声室验收通过、不等于 bit-exact、不等于 ≥90dB 承诺达标——那些是 R3 / CRC 门 / SPL 解冻的独立事项。
