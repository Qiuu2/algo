### 2.2 失败教训 (Failure Lessons)

#### Fail-001: 网格过粗导致共振频率偏移
```yaml
case_id: "FAIL-2023-003"
project: "Subwoofer_15in"
type: "箱体共振仿真"
status: "failure_resolved"

failure_description: "仿真预测箱体面板共振频率为185Hz，实测为148Hz，偏差25%"

root_cause: "网格过粗（λ/4 at 200Hz），未能捕捉薄板弯曲模式"

resolution:
  action: "网格细化至λ/8，面板区域增加边界层"
  result: "共振频率仿真值152Hz，偏差降至2.7%"
  compute_cost_increase: "4x"

lessons_learned:
  - "薄板弯曲振动必须满足 λ/6 最小网格要求"
  - "箱体面板区域需要局部网格细化"
  - "共振问题必须用特征频率分析验证模态形状"
  - "后续所有箱体仿真强制使用 λ/8 网格标准"

prevention_checklist:
  - "薄壁结构网格检查: 厚度方向≥3层单元"
  - "特征频率分析作为标准步骤"
  - "与解析解对比确认弯曲波速"
```

#### Fail-002: 空气吸收参数遗漏
```yaml
case_id: "FAIL-2023-007"

  - "空气吸收系数: α ≈ 0.005-0.05 dB/m @ 1kHz-8kHz (标准条件)"

prevention_checklist:
  - "传播距离>20m → 启用空气吸收"
  - "频率>4kHz → 启用空气吸收"
  - "环境参数记录: 温度/湿度/气压"
```

## 3. 网格划分策略经验 (Meshing Strategy Knowledge)

### 3.1 按问题类型推荐网格策略

| 问题类型 | 推荐方法 | 网格密度 | 特殊处理 | 预估DOF(每m³) |
|----------|----------|----------|----------|---------------|
| 低频音箱箱体 (<500Hz) | FEM四面体 | λ/6-λ/8 | 面板细化 | 10K-50K |
| 全频音箱 (至20kHz) | FEM四面体 | λ/8 | 号角/端口细化 | 100K-500K |
| 阵列近场分析 | FEM四面体 | λ/8-λ/10 | 单元间隙细化 | 500K-2M |
| 阵列远场辐射 | BEM + 远场 | λ/6 | 表面网格 | 50K-200K |
| 房间声学(射线) | 射线追踪 | N/A | 表面网格用于反射 | 10K-50K(表面) |
| 声波导管(号角) | FEM + BEM | λ/8-λ/10 | 喉口极细化 | 200K-1M |
| 微声学(手机) | FEM热黏性 | λ/10-λ/20 | 边界层必须 | 1M-10M |

### 3.2 常见网格问题与解决方案

| 问题现象 | 可能原因 | 解决方案 | 预防方法 |
|----------|----------|----------|----------|
| 求解不收敛 | 网格质量差（Skewness<0.1） | 重新划分，局部修复 | 网格质量检查自动化 |
| 结果振荡 | 波长内单元不足 | 全局或局部加密 | 频率-网格自动关联 |
| 内存不足 | DOF过多 | 降阶模型/混合方法 | 前期DOF估算 |
| 边界层分离 | 边界层网格冲突 | 调整边界层参数 | 几何清理后检查 |
| 多尺度几何 | 大小特征差异大 | 局部细化+过渡区域 | 几何简化分级 |

| 问题规模 | 内存(GB) | 时间(分钟) | 最佳参数 | 备注 |

| <100K DOF | 4-8 | 1-5 | 默认 | 小规模最优 |
| 100K-500K | 16-32 | 5-20 | 核数=8 | 中等规模 |
| 500K-1M | 64-128 | 20-60 | 核数=16 | 大规模 |
| 1M-2M | 128-256 | 60-180 | 核数=32, 分布式 | 超大规模 |

**MUMPS调优参数：**
```
out-of-core: 内存<64GB时启用
parallelization: 核数=2-4 per 100K DOF
preordering: METIS (一般) / AMD (结构问题)
scaling: 启用（改善矩阵条件数）
```

### 4.2 迭代求解器 (GMRES)

| 场景 | 预处理器 | 迭代次数 | 收敛性 | 建议 |
|------|----------|----------|--------|------|
| 声学FEM | GMG几何多重网格 | 50-200 | 好 | 推荐大规模问题 |
| 声学FEM+PEC | SA-AMG代数多重网格 | 100-500 | 中 | 复杂几何 |
| BEM | 快速多极子(FMM) | 50-300 | 好 | BEM标配 |

**收敛判据：**
- 相对残差 < 1e-6
- 绝对残差 < 1e-12
- 最大迭代次数：1000（默认500可能不足）

## 7. ITC 定向音柱项目 — d=55mm 阵列基线（Sprint 3 实战记忆，2026-05-27；PF-8 重审 2026-05-29）

> **唯一自研基线（DEC-S3-GEOM-01，LOCKED）**：N=16 / **d=55mm[L1拆机实测]** / L=825mm。
> 历史上出现过的 **d=30mm / L=450mm 系视觉估测错误值，已全部作废**，记忆中不再保留为"基线"或"对比基线"。

### 7.1 基线几何（竞品工程基准，拆机真值）
- **N = 16 / d = 55mm[L1拆机实测] / L = 825mm**（声学孔径 (N-1)d；注意 880mm 是箱体物理长度，非孔径）
- 驱动：8 路 A/B 关于中心**对称串联** → **broadside-only**（无电子偏转，DEC-S3-DSP-03 接受）
- 加权：Dolph-Chebyshev -20dB。**8 对称对加权与 16 独立元结果完全相同**（Dolph 本身满足对称性，对称对约束不损失 BW/SLL）

### 7.2 基线指向性数据（sweep_d55.py 实算，点声源远场阵因子）
- **BW@1kHz = 29.3°** ——无需任何超指向即满足 ≤30° 规格（d=55mm 实测基线值；d=30 为错误估测，不作对比基线）
- **BW@2kHz = 14.5°** ——远超 ≤30° 目标
- 栅瓣（d=55mm[L1/几何]）：**500Hz–5kHz 安全**（d/λ<1.0）；半波长判据 d=λ/2→**3118Hz**，严格 broadside 判据 d=λ→**6236Hz**；**6.24kHz 起栅瓣峰 0dB（与主瓣等高）、带外泄漏 78–89%[L2 仿真]**。应对：强指向规格上限由 8kHz 降级为 **6kHz(对内)/5kHz(对外保守)**（方向1，CTO 采纳），6.2–8kHz 落在降级规格之外（如需覆盖须缩间距/分频段）

### 7.3 关键修正与教训
- **PF-8 教训（视觉估测当实测，2026-05-29）**：d=30mm/L=450mm 曾被当作"基线"在多份活文档传播，实为**目测/视觉估测值**，与拆机游标卡尺实测 d=55mm/L=825mm 相差近 1.83× 孔径。教训：**只有带 [L1拆机实测] 标记的几何值才可作基线**；视觉/照片估测必须标 [估测] 且不得进入设计输入；措辞红线——仿真值绝不可称"实测"。栅瓣临界、孔径、BW 均强依赖 d，估测错误会连带污染规格上限与工装尺寸。
- **竞品真值 = 16/55（L825），不是 Sprint2 反推的 ≈20/35**。反推存在多解性，RMS 仅供 PRD 对标参考，**实测/拆机数据优先**（详见 [[critic-memory]] LESSON-006）。
- 点声源模型 **≥2kHz 高频置信度低**：4kHz 仿真 7.2° vs 竞品 BW≈19.1° [L2/L3 4点插值派生·F-AC-01 已撤回·非 L1 实测]（箱体衍射/单元物理效应主导，阵因子模型失效；竞品仅实测 SPL，该 BW 系从 SPL 插值反推、已撤回）。高频 BW 须以消声室实测为准。
- **SPL 预测为理论上限、低置信度**（~117dB，比竞品同款硬件实测高 6-14dB，DEC-S3-DSP-05）；最终 SPL 待真实 T/S 参数（送测中 2-4 周）+ 消声室实测。

### 7.5 JY/T 表9 国产标准合规检查（TASK-STD-A，2026-05-29，三轨一致到 0.001dB）
**口径**：SPL差值 = −20log10|AF(θ)/AF(0)| = θ处衰减量（≠BW；同一 d=55 阵因子，Dolph-20，isotropic）。
**两前置陷阱裁定**：
- 陷阱1（平面口径）：线阵指向性平面唯一(阵列轴平面)。假设我方波束平面=表9水平面 → **前提=音柱必须水平安装**；竖装则水平面近全向、表9水平要求全失效。**不 ESCALATE**（可对齐，非根本错配），但安装朝向作为条件项报 CTO。
- 陷阱2（180°对称）：isotropic 阵因子前后对称 → **AF(180°)=AF(0°) → 180°衰减=0.0000dB（artifact，已三轨验证）**。后向抑制全靠阵元/障板（现模型零此能力）→ **180° 全部4点不可评，禁报误导数字，待 COMSOL障板+消声室实测**。
**12点结论**（30/90 实算）：可评8点中 **7点达一级，唯 2kHz/90°=23.01dB 仅二级（距一级缺1.99dB）**。
**R8 命中**：500Hz/30° = **5.857dB → 一级裕量 +0.857dB ≈ +0.86dB**（与CTO暗示完全吻合，反证平面口径假设成立）；此为全表最薄一级点。
**短板**（对照CTO提示竞品短板500/180°+4000/30°）：500Hz/180°=不可评(P0最关键待验，门限12/10/8)；4000/30°一级但裕量仅+1.01dB且高频PF-6前向偏乐观→实测可能跌级；2kHz/90°仅二级。
产出：`sprint3/audit/standard_compliance_check.md`、`std_table9_compliance.py/.csv`。
