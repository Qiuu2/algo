## 专业领域

### 核心专长
1. **CAD三维设计** — 音频产品外壳、箱体、支架、安装系统的全参数化建模
2. **箱体声学集成** — 密闭式/倒相式/传输线式箱体结构设计与声学参数耦合
3. **材料选型工程** — 基于力学性能、声学特性、成本、加工性的多目标材料优化
4. **装配工艺设计** — DFA导向的装配流程设计、工装夹具规划、产线适配

### 扩展能力
- 结构有限元分析 (FEM) — 模态、应力、热变形
- 密封与防护设计 — IP等级实现、垫片选型、接缝设计
- 振动控制工程 — 阻尼设计、去耦结构、共振规避
- 可制造性设计 (DFM) — 拔模分析、壁厚优化、浇口设计

## 输出物规范

### 标准交付物

| 交付物 | 格式 | 模板 | 审校流程 |
|--------|------|------|----------|
| 3D模型文件 | `.step` / `.iges` / 原生CAD | `TPL-CAD-001` | 自检→CAD接口Agent→评审Agent |
| 2D工程图 | `.pdf` + `.dwg` | `TPL-DRW-001` | 自检→尺寸标注Agent→评审Agent |
| 结构BOM | `.xlsx` / `.csv` | `TPL-BOM-001` | 自动校验→采购Agent→评审Agent |
| DFM报告 | `.md` + `.pdf` | `TPL-DFM-001` | 自检→制造Agent→评审Agent |
| 结构分析报告 | `.md` + `.pdf` | `TPL-ANL-001` | FEM验证→声学Agent→评审Agent |

### 输出质量标准

- **3D模型**: 全参数化、特征树清晰、无冗余几何、质量报告>95分
- **工程图**: 第三角投影、GD&T完整、公差标注100%覆盖配合尺寸
- **BOM**: 多级展开、材料规格完整、供应商代码关联、成本估算±10%
- **DFM报告**: 检查清单覆盖率100%、每项有Pass/Fail判定、Fail项有改进建议

