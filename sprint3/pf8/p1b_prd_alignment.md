## 2. 安装可行性评估（825mm vs 450mm，定性 [L3]）

### 2.1 尺寸变化事实
- 阵列声学孔径：450mm → **825mm（+72%）**[L1：(N−1)×d=15×55mm]。
- 产品整机外形长度将 ≈ 850–900mm[L3 推断，含端部余量/封堵/安装耳]，属"接近 1 米的细长立柱"形态。
- 对比参照：典型壁挂音柱产品长度区间 500mm–1200mm，825mm 处于**行业常规音柱长度带内**，非异常尺寸。

### 2.2 分场景影响（定性）

| 场景 | 安装方式 | 可行性 [L3] | 关注点 |
|------|---------|------------|--------|
| **博物馆讲解** | 壁挂（展墙/展柜上方）/ 立柱（落地展位）| **可行** | 视觉接受度需关注：825mm 细长柱在展陈环境较明显，建议配深色/隐蔽化外观；覆盖距离更窄更聚焦，利于"单展品定向讲解"，对长度增加反而有正收益 |
| **车站广播** | 吊装（站台顶）/ 壁挂（立柱/墙面）| **可行，最有利** | 车站空间尺度大，825mm 视觉无碍；更长孔径 = 更窄波束 = 更远定向覆盖距离，匹配站台分区广播需求；吊装须确认结构承重与吊点间距 [待结构确认] |
| **商场分区广播** | 吊装（吊顶）/ 壁挂（柱面）| **可行** | 商场层高足够，825mm 吊装可接受；分区隔离度因孔径增大而提升（窄波束减少串扰）；视觉上需与吊顶装修协调，建议提供吊装支架选配 |

### 2.3 物理性能侧收益（非纯负担）
- 孔径 +72% → 同频束宽更窄、定向性更强（BW@2k 14.5° vs 竞品 18.1°[L2]），**覆盖距离与分区隔离度提升**，对三类场景的"定向广播"核心诉求是**正向收益**。
- 代价集中在：① 机械尺寸/重量增加（安装承重、运输包装）；② 视觉体量增大（博物馆敏感）；③ 6.2–8kHz 栅瓣（已规格降级管理，与安装无关）。

### 2.4 定性结论
**825mm 对三类目标场景均可接受**，无场景因长度变化而不可安装。最需要管理的是**博物馆视觉接受度**（外观隐蔽化设计）与**吊装/壁挂的机械承重与安装点间距**（[待结构/硬件确认]）。建议结构 Agent 在出图时同步给出三种安装方式（壁挂/立柱/吊装）的支架与承重方案。

## 4. PCB 节奏影响评估（CTO 关注项）

### 4.1 推理链
**问题**：d=30→55mm 是否影响 Sprint 3 自研 PCB（ADSP-21569 主控板）节奏？

**事实依据**（`sprint3/hardware/selfdev_bom.md` v2 / `transition_board_design.md`）：
- 自研 PCB = **电子主控板**：ADSP-21569（DSP）+ ADAU1962A（8ch DAC）+ ACM3128A/TAS5825M（8ch 功放）+ 电源树 + TDM-8 互连，**4 层自研板**。
- 阵元间距 d 是**阵列承载结构/工装**的几何参数（决定 16 只喇叭的物理排布间距）。
- 主控板与喇叭阵列之间通过**接口/连接器/出线端子**连接（selfdev_bom 第 167 行"接口/连接器"），**喇叭按 8 路功放输出走线到端子**，端子到各喇叭单元的实际物理间距由**承载结构**决定，**不在主控 PCB 板内布局**。

### 4.2 是否有"阵元接口布局依赖 d"的情况？
- 核查 `selfdev_bom.md`：PCB 上为 8 路功放输出端子（A/B 串联拓扑，8 路 15Ω），**输出端子数量=8，与 d 无关**；端子在板上的布局是电气走线/散热驱动，**非按阵元 30mm/55mm 间距排布**。
- 即使存在"板载喇叭出线排针按阵元间距排列"的设计（本项目**未采用**——采用连接器/端子 + 外部线束到承载结构），d 变化也只改外部线束/承载结构，不改板内核心电路（DSP/DAC/功放/电源/TDM）布局与器件选型。
- 算力侧已确认：**d 不影响 MMAC/算力裕量**（`budget_calc.py` 注、`dsp_design.md`"d 不影响算力"）。

### 4.3 结论：**PCB 节奏不受影响（否）** [L3]
**理由**：
1. 自研 PCB 是电子主控板（DSP+DAC+功放+电源+TDM），其电路设计、器件选型、layout 由**电气架构**决定，与阵元几何间距 d **解耦**。
2. d=55 影响的是**阵列承载结构/工装**（喇叭物理排布），属结构域，已在 §3 工装重做覆盖。
3. 主控板与阵列经连接器/端子/线束连接，d 变化只改外部线束与承载件，**不改板内布局**。
4. 输出通道数（8 路）由 A/B 串联拓扑决定，与 d 无关；d 变化不改通道数、不改 DAC/功放选型。

> **因此不满足 CTO 的 ESCALATE-E 条件（"d=55 影响 PCB 节奏"）→ 不触发 🚨ESCALATE。**

### 4.4 边界说明（诚实标注）
- 上述为 [L3] 工程推断，基于 selfdev_bom v2 / transition_board_design v2 的架构（主控板 + 外部线束到承载结构）。
- **唯一需硬件 Agent 最终确认 [待硬件确认]**：自研 PCB v0.1 原理图/layout 冻结时，确认喇叭出线**确为连接器/端子方案**（而非板载按阵元间距排布的排针）。若硬件最终改为"板载阵元排针按 d 间距"的非常规方案，则需回到本节重评——但当前 BOM/架构文档无此设计意图。

## 5. 内部对齐说明（产品尺寸 450→825 的由来 + 场景影响，供 CTO 对外沟通用）

> 本节为简短内部对齐材料；**是否对外由 CTO 决定**。措辞已规避对外不可披露的低置信度数字（如绝对 SPL）。

### 产品阵列尺寸由 450mm 调整为 825mm —— 内部说明

**一句话**：经拆机实测确认，量产基线阵元间距由早期估测的 30mm 修正为实测 55mm，阵列声学孔径相应由 450mm 调整为 825mm（+72%），这是**基于实测真值的工程修正，非设计变更扩张**。

**由来（内部口径）**：
- 早期（Sprint 2）阵元间距 30mm 系基于竞品**视觉估测**，未经物理测量。
- Sprint 3 实物拆机测量确认真实间距为 **55mm**，孔径随之为 (16−1)×55 = 825mm。
- 公司决策（DEC-S3-GEOM-01）将自研基线与实测真值统一，消除"两套几何并存"的协同成本。

**对产品/场景的影响（内部口径）**：
- **正向**：更长孔径带来更窄、更聚焦的定向波束，**定向覆盖距离与分区隔离度提升**，更契合博物馆单展品讲解、车站/商场分区广播的核心诉求。
- **代价**：① 整机更细长（约 0.85–0.9m），博物馆等视觉敏感场景需配隐蔽化外观；② 高频（6.2kHz 以上）出现栅瓣，**强指向规格上限由 8kHz 收敛至 6kHz（对内）/ 5kHz（对外保守）**——人声讲解与广播主能量集中在 ≤5kHz，**对语音清晰度与定向效果无实质影响**。
- **安装**：壁挂/立柱/吊装三种方式对三类场景均可行，无场景因尺寸增大而不可安装。

**对外沟通建议（若 CTO 决定对外）**：
- 强调"定向更强、覆盖更聚焦"的**正向卖点**，淡化尺寸增大。
- 强指向频段对外按 **5kHz 保守口径**表述，避免 6–8kHz 栅瓣区被外部测试质疑。
- **不对外披露**绝对 SPL 预测值（117.1dB 为理论上限、低置信度，PRD §7.3 已标"不可对外"）。
- **永久边界提醒**：任何对外承诺或市场需求若回头要求 6–8kHz 强指向，将触发几何重议（SC-S3-GEOM-01），属重大变更，需 CTO 介入。
