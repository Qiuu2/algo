## 0. 四个数据源 = 四个触发器

| 触发器 | 解锁什么 | 预计到位 | 前置依赖 |
|--------|---------|---------|---------|
| **T1 EZKIT 到货** | 命门 R1 + 定点/延迟/时序全部上板项 | 海外周期（已下单）| 树形C✅；转接板 PO（卡《6项未知量报告》签署）|
| **T2 T/S 参数回报** | SPL 重做 + 互耦 + 阻抗匹配 | 2–4 周（送测中）| 无 |
| **T3 消声室档期** | BW/栅瓣/超指向/balloon/SPL 实测对标 | 需预约（按 6 周倒推）| 样机（竞品壳验证线 / 自研件）|
| **T4 COMSOL 流程** | 有限障板衍射 + 完整公差MC | 桌面工具，需建模工时 | 几何已定 d=55 ✅ |

## 1. T1 — EZKIT 到货后（命门，优先级最高）

```
T1 EZKIT 到货
  ├─[需转接板] 转接板 PO ← 卡《6项未知量报告》签署（U1-U6）
  │     └─ ⚠️ 转接板 PO 是 T1 真正启动前的闸门，须先解
  └─[EZKIT 本体即可启动，不依赖转接板]
        ├── ★ R1 命门：树形C 移植 EZKIT + CCES cycle-counter 实测真实 MCPS
        │        → 含 cache/取模/中断开销；判 ≥10× 关闭线；定 21565 vs 21569（DEC-S3-PROC-01 解冻）
        ├── PF-4 L1：SHARC 定点 SNR 逐bit（浮点 vs 定点上板对比）
        ├── PF-2 L1：端到端延迟硬件确认（真实 DAC 群延迟）
        ├── 节点①饱和：真实节目素材跑满量程，标定触发率 + −4.8dB headroom 取值
        ├── WCET / MCPS 峰值（取模/cache miss/中断，验 WCET<帧周期1.33ms）
        └── DMA/中断时序 / SPORT underrun（SB3 子带率下裕度）← 还需 U4 拆机 TDM 数据
```
**关键路径**：转接板 PO 解锁 → EZKIT 上电 → R1 MCPS（命门）→ 芯片选型解冻。

## 2. T2 — T/S 参数回报后

```
T2 T/S（BL/Re/Fs/Qts）到位
  ├── PF-3/R3：绝对 SPL 重做（含 BL 电声模型，替换 117dB 占位）
  ├── 阵元互耦（55mm 2寸喇叭近距，需单元参数）
  └── 8路 A/B 串联 vs 功放阻抗匹配（串联≈15Ω，失配对幅相一致性）
```

## 3. T3 — 消声室档期后

```
T3 消声室（需样机）
  ├── ★ R5/DEC-S3-DSP-06：实测 BW@1k → 裁定 1kHz 超指向去留（>30°启用 / 达标关闭；仿真29.28°压线0.72°）
  ├── R6：栅瓣 6.2–8kHz 实测对标（验证规格降级6k是否够，方向2阵元指向性补偿评估）
  ├── R4 复核：BW@2k=14.5° 仿真 vs 实测（≤30°）
  ├── 垂直面/俯仰面 balloon（安装高度/覆盖距离）
  ├── PF-3：绝对 SPL 消声室校准（对标竞品0° 106-111dB）
  └── PF-6：高频精确指向性（替 isotropic，4kHz竞品反常DEC-S3-AC-01顺带测）
```

## 4. T4 — COMSOL 流程后

```
T4 COMSOL（建模工时，几何已定）
  ├── 有限障板/箱体衍射（825mm有限孔径+边缘衍射，疑似竞品4kHz反常主因）
  └── 公差MC 真实障板版（替 P0-1 cosine 近似乐观下界，SLL产线散差真值）
```

## 5. 启动协议（待命 → 激活）
1. 任一 T1-T4 到位 → PM **立即 ESCALATE 通报 CTO**（不自动启动）。
2. CTO 定节奏后，PM 按上表派单（teammate）；命门 R1 优先。
3. 每项产出沿用 PF 教训：数字标 L1、过 Critic C1–C6、L1 实测回填对应 DEC（PF-1~PF-8 的"待 L1"项逐条关闭）。
4. R1 出真值后回填 DEC-S1-004/DEC-S3-PROC-01（芯片选型终局）。
