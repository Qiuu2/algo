## 二、SPLo 82.16dB vs 通带 ~90dB 的 8dB 差 —— 归因（PM 已验证，非口径错误）

**结论：8dB 不是口径错配，是 SPLo（活塞区参考效率）与带内实测灵敏度的正常物理差。**

- **SPLo = 82.16dB = 半空间参考效率（1W/1m）= 低频活塞区渐近线**，纯由 T/S 推导：
  η₀ = (ρ₀/2πc)·(BL²·Sd²)/(Re·Mms²)。**PM 用图2 值回算**：BL=2.03, Sd=1.452e-3 m², Re=7.6, Mms=0.783e-3 kg → η₀≈**0.102%**（=图2 No=0.103% ✓），SPLo=112.1+10log₁₀(η₀)=**82.19dB**（=图2 SPLo 82.16 ✓）。**自洽，读数可信。**
- **通带 ~90dB（图1 200Hz–5kHz）是带内实测**：小口径单元（Fs=236Hz、Qts=1.46）在谐振以上响应**自然抬升**（质量控制区→分割振动区上扬），高于活塞区渐近线 SPLo 属**正常**，量级（+8dB）对 2 寸单元合理。
- **对 X2 的处置**：工作频段 1k–8k 应取**图1 实测曲线逐频灵敏度（~90dB 区）**，**不取**单一 SPLo=82.16。但**绝对电平须靠 M1 测距 + M2 电压锚定**——故 8dB 归因虽解释了"差从哪来"，**X2 仍卡 M1/M2 才能定绝对值**。
- ⚠️ **反推废止**：旧 117 占位地基用的 88dB/1W 灵敏度[L4 估测]，已被本实测体系**整体推翻**（SPLo 82 / 带内 90 都不是 88）→ 117 退役，不作 X2 基准。
