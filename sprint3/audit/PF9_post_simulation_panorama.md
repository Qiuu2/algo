### C.2 风险（Critic 主写，基于 PF-9 教训）

> 下列每项均标等级。凡可能复发 PF-9（SPL→BW 反推 / 撤回值复活 / 等级冒充 / 定性滑成定量）者 **逐一点名为 BLOCKER 红线**，对比方案触碰任一条即整体打回。

**R-C2-1【BLOCKER 红线】从竞品 PDF 反推 BW = PF-9 同款病，绝对禁止。**
竞品 §7 只实测 6 个角度 SPL（0/30/60/90/150/180°），栅格粗（30° 步进，主瓣区仅 0°/30° 两点）且 **SPL≠BW**（SPL 是某角度声压，BW 是 −6dB 全角张角，二者非同一物理量）。任何"从 6 点 SPL 插值/拟合主瓣再读 −6dB 全角"的动作，正是 F-AC-01 撤回的 14.9/19.1/16.0° 的生成方式（PF-9 病根）。**任何形式的"竞品 BW"——无论叫实测、插值、估算、参考——一律 BLOCKER，对比表禁止出现竞品 BW 列。** 我方 BW[L2] 只能与未来 **消声室 L1 实测 BW** 比，不与竞品 PDF 比。

**R-C2-2【BLOCKER 红线】我方 117dB[L4 占位] 与竞品 0° 106–111dB[L1] 直接比 = L4 冒充可比量。**
我方 117dB 是无 BL 电声模型的理论上限占位（PF-3，DEC-S3-DSP-05 已判不可对外），竞品 106–111dB 是消声室实测[L1]。两者放进同一对比即把 L4 占位伪装成与 L1 可比的工程量，传播假精度（典型等级混乱）。**绝对 SPL 维度在 R3（T/S + BL 重做）完成前，禁止任何我方 vs 竞品 SPL 数值对比**；当前只能登记"待重做"。任何"我方比竞品高 6–14dB"之类表述 = BLOCKER。

**R-C2-3【MAJOR→若违则 BLOCKER】归一化方向图"定性吻合"被滑成"定量背书"。**
推荐①的归一化相对衰减对比，受两重限制只能定性：(i) 角度栅格不一致——我方连续/竞品 30° 步进，无法在同角度逐点对齐；(ii) 高频 2–4k 阵元 iso 失真（PF-6），我方该频段方向图本身低置信。风险是读者把"趋势同向"误读为"仿真已被实测验证/dB 级吻合"，进而当成 Gate 背书。**红线：归一化方向图对比只许出"方向性吻合/部分/背离"定性结论，禁出任何"偏差 ≤X dB 合格""仿真已验证"字样；2–4k 必须显式标 iso 失真、不作证据。** 滑向定量背书即升 BLOCKER。

**R-C2-4【BLOCKER 红线】撤回值复活 / 张冠李戴值再引用。**
14.9/19.1/16.0°（F-AC-01 SPL 插值反推）、19.2°（ITC 自仿 N16/d25 行张冠李戴）、d=30 全套（L0 目测，PF-8 作废）一律不得作任何对比/派生/"参考值"依据；引用其字面值时必须带"已撤回/作废"警示（C7）。任何把这些值当现行竞品指标复活的对比 = BLOCKER。

**R-C2-5【BLOCKER 红线】后向/前后比/DI 的伪同口径对比。**
我方零后向仿真（阵因子点源模型不含箱体后辐射），竞品有 150/180° SPL[L1]。任何"用竞品后向 SPL 反推我方前后比/DI""单边数据凑对比"= 制造同口径假象，BLOCKER。后向只能登记为单边维度（竞品有/我方无/未来补基线），不参与对比。

**R-C2-6【MAJOR】4kHz 反常对比越界到 BW 数字。**
4kHz 反常核验只能停在 **SPL 衰减形态**层面（竞品 4k 各角衰减是否比 2k 更慢）。一旦把"竞品 4k 更宽"落成任何 BW 数字（如复活 19.1°），即同时踩 R-C2-1 与 R-C2-4，升 BLOCKER。该项只能产出"实测背书动机（指向 COMSOL 障板建模）"，不得产出"实测结论"。

### C.3 安全可执行对比推荐

> 只列安全项（绕开 PF-9/F-AC-01/L4 占位）。每项给：对比方法 + 输出格式 + Critic 复核重点。

**推荐①：0° vs 各角 归一化 SPL 衰减模式对比（同口径，定性为主）**
- **对比方法**：我方仿真方向图取 0° 为基准，输出 0°−{30/60/90} 相对衰减(dB)；竞品 §7 SPL 同样 0°−各角归一化为相对衰减。两侧均为**归一化相对衰减**(无绝对 SPL 介入)，按频点(250/500/1k；2-4k 仅作存疑标注)对位比对衰减趋势。
- **输出格式**：表格(频点 × {30/60/90°} × {我方归一化衰减 / 竞品归一化衰减 / 趋势是否同向})+ 一句定性结论(吻合/部分/背离)。**不出 dB 级"偏差合格"判定**，只出方向性吻合度。
- **Critic 复核重点**：(a) 两侧是否均已归一化、确无绝对 SPL(L4)混入；(b) 角度栅格不一致是否已声明为定性限制；(c) 2-4k 是否已标 iso 失真、未当定量证据；(d) 措辞未把仿真称实测。

**推荐②：4kHz 角度分布反常的实测背书强度（定性，仅 SPL 域，绝不推 BW）**
- **对比方法**：仅看竞品 §7 SPL：4kHz 各角衰减形态 vs 2kHz 各角衰减形态，定性判断 4k 是否角度衰减更慢(=主瓣更宽的间接迹象)。与我方阵因子"4k 比 2k 更窄"方向对照，确认阵因子点源模型在 4k 失效、需箱体衍射(COMSOL 声学2)解释。
- **输出格式**：一句定性陈述 + 证据强度评级(弱/中)；**严禁出现任何 4kHz BW 数字**，只描述 SPL 衰减形态。结论指向"需 COMSOL/消声室实测裁定"。
- **Critic 复核重点**：(a) 全程停留在 SPL 衰减形态、零 BW 派生；(b) 未引用任何已撤回竞品 BW(14.9/19.1/16.0/19.2°)；(c) 措辞为"实测背书动机"而非"实测结论"。

**推荐③：后向辐射(150°/180°)——单边维度，标注非对比 / 未来补基线**
- **对比方法**：**不做对比**。仅在报告中登记为"竞品 PDF 独有维度(有 150/180° SPL[L1])、我方零后向仿真"，列为我方未来基线缺口(补后向仿真 + 完整 balloon)。
- **输出格式**：单边维度登记条目(竞品有/我方无/未来补)，**不出任何前后比/DI 数字**。
- **Critic 复核重点**：确认未从竞品后向 SPL 反推任何我方指标、未构造伪前后比。

1. **当前可信仿真覆盖度（够进 PCB 前评估，不够拍不可逆）**：真正 L1/L2 可信完成 = 声学 **A（水平面 BW，1k=29.28°/2k=14.5°）/ G（旁瓣-20dB+栅瓣 6236Hz）/ 栅瓣 criterion** + DSP **①（MMAC 预算）/ ②（滤波器组阻带）/ ②+（树形 C 重建 182dB）/ ④（定点化桌面 L2）/ 节点①溢出修复**；命门 **⑤算力裕量桌面实算 17×/33×**（>10× 结论可信，但芯片选型待 L1）。占位/部分/未做 = SPL（L4 占位）、方向图（仅水平面）、超指向去留（待消声室）。**几何已统一 d=55（PF-8）、定点已桌面闭环（PF-4）、撤回值已全库清扫（PF-9/E-NEW-3，C7 PASS）。**

2. **最大缺口 = ①数据缺（非桌面工作量）**：唯一 **P0 决策阻塞 = ⑤/R1 EZKIT cycle MCPS 实测**（卡 EZKIT，定 21565 vs 21569）。其余 P1 重缺口全卡外部数据：SPL 重做卡 T/S、超指向/栅瓣/balloon 卡消声室、有限障板衍射卡 COMSOL、温漂卡 datasheet。桌面 P1 仅 **D15 Dolph C 移植 + D8 量化噪声谱** 值得上板前置补；其余桌面项性价比低于已完成的 PF-1~9。**真瓶颈是外部数据到位，不是缺人手。**

3. **竞品对比任务推荐：有限做（仅 C.3 三项定性，零定量/零 BW 派生），核心价值是为 COMSOL/消声室"喂动机"而非"出结论"**。竞品 PDF 只有 SPL[L1]、无 BW——可安全对比的**只有归一化 SPL 衰减模式（定性）+ 4kHz 反常的 SPL 域背书（指向 COMSOL 障板建模）+ 后向辐射单边登记**。**❌ 不做**：BW 对比（SPL→BW=PF-9 BLOCKER）、绝对 SPL 对比（L4 占位 vs L1）、前后比/DI（我方无后向数据）。**结论**：对比能产出"方向性吻合度 + COMSOL/消声室必要性背书"，**但产不出"仿真已被实测验证"的定量结论**（竞品几何 d=55 与我方同、但只有 SPL）。建议**做**——成本低、为后续外部数据任务定向；但 CTO 须明确其定位 = 定性背书，不是 Gate 级验证。
