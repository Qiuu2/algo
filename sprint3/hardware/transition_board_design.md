> **v2 修订摘要（BLOCKER F-X01 闭环）**：
> A/B 串联对称驱动后，末端真实驱动通道 = **8**（竞品 ACM3128A：4 片 × 2ch BTL = 8ch；DSP 已按 8 通道 TDM 设计 = 8 slot / BCLK 12.288MHz / FSYNC 48kHz）。
> 原 v0.1 选 16ch ADAU1966A（16 slot / BCLK 24.576MHz），接 8ch DSP 输出会导致 **TDM 帧错位、DAC 输出错位/静默** → 已纠正。
> 本版改动：① DAC 改为 **8 通道方案（ADAU1962A，用其中 8ch）**，放弃 ADAU1966A 16ch 首选；② TDM 统一 **8 slot / BCLK 12.288MHz / FSYNC 48kHz**，与 DSP 对齐；③ 电平分压网络按 ADAU1962A 满幅重算；④ 措辞澄清：竞品器件按公开 datasheet 采购，非复用竞品成品板/固件。

## 0. 总体架构

### 0.1 信号链路（以竞品模拟输入功放为基础情形）

```
EV-21569-EZKIT
    │
    └─ SPORT0 (TDM-8, 48kHz, 32bit/slot, BCLK 12.288MHz)
         │
         ▼
   [转接板 — ADAU1962A 8ch DAC（用 8 通道）]
         │ 8路单端模拟输出（重建LPF后）
         ▼
   [电平/阻抗匹配网络 × 8]
   [串联衰减 R-R 分压 + AC耦合电容]
   [RC+TVS 限幅保护（防过驱动烧竞品功放）]
         │
         ▼（8通道 → 竞品8通道输入）
   竞品 ACM3128A 功放（4 片 × 2ch BTL = 8ch输入）
   [灵敏度 0.246 Vrms，输入阻抗 20kΩ]
         │
         ▼
   竞品喇叭阵列（A/B串联，8路 × 2只 = 16只，15Ω/ch）
```

> **8 通道一致性链路**：DSP（8 slot TDM）→ DAC（8ch）→ 功放（8ch BTL）→ 喇叭（A/B 串联 8 路 × 2 只 = 16 只）。全链路通道数 = 8，无失配。

**注意**：竞品接口（模拟/数字）必须拆机确认（U1）；若为数字 TDM 接口，可省掉 DAC，方案简化为数字直连（见 1.3 节情形②）。

## 1. 关键设计模块详解

### 1.1 DAC 选型：8 通道方案（ADAU1962A）

#### 通道数裁决（BLOCKER F-X01 闭环）

末端真实驱动通道 = **8**（A/B 串联对称驱动；竞品 ACM3128A 4片×2=8ch）。DSP 已按 **8 通道 / 8 slot TDM** 输出。因此 DAC 必须为 8 通道方案，**放弃** v0.1 的 16ch ADAU1966A（16 slot 时钟会与 8ch DSP 输出帧失配）。

| 项目 | 选型 | 依据 |
|------|------|------|
| **转接板首选 DAC** | **ADAU1962A，用 8 通道**（CTO 任务书指定）| 对齐末端 8 通道；TDM/SPORT 接口匹配 21569；只需单片 |
| 通道数说明 | ADAU1962A **实际为 12 通道 [待核实]**（见下方告警），8ch 验证仅用其中 8 路，ch9~ch12 静默/不引出 | 见 [待核实] 项 |

#### 关键告警 [待核实]

> **[待核实] ADAU1962A 实际通道数与 TDM/slot 支持**：
> - CTO 任务书称 ADAU1962A 为 8ch；但 ADI 官方资料（sprint3_retrofit_assessment.md 引用）显示 **ADAU1962A = 12 通道** DAC（24bit/192kHz）。
> - 无论 8ch 还是 12ch，均能覆盖本方案所需的 **8 通道**（12ch 时仅用前 8 路）。
> - **必须查 ADI 官方 Datasheet 确认**：① ADAU1962A 确切通道数；② 是否支持 TDM 输入、最大 slot 数（需 ≥8 slot）、可配置为 8 slot / 32bit/slot；③ 单端满幅输出电压（电平网络依赖此值，见 1.1 输出电平匹配）。
> - 若 Datasheet 确认 ADAU1962A 不支持 8-slot TDM 或不满足输出电平要求，备选 8ch DAC：**PCM1690（TI，8ch，DAC，TDM）** 或 **ADAU1966A 配置为 8 slot 模式**（若 ADAU1966A 支持 8-slot 子集配置且 DSP 输出 8 slot，亦可，但需软件严格对齐 slot 数 = 8，不可用 16 slot）。

**ADAU1962A 关键规格**（参考 ADI 资料，[估算/待核实]）：
- 通道数：**12ch [待核实]**（任务书称 8ch；本方案仅用 8 路）
- 接口：TDM/I2S，支持 SPORT 连接（[待核实] 最大 slot 数 ≥8）
- 输出方式：差分/单端模拟输出（[待核实] 是否需外接 I/V，部分 ADAU 系列内置输出缓冲）
- 满幅输出电压：约 **±1.0Vrms 差分 / 约 0.5Vrms 单端 [待核实]**（须查 Datasheet）
- 供电：AVDD 3.3V / DVDD 3.3V（[待核实]）
- 封装：LQFP / LFCSP（[待核实]）

#### 输出电平匹配设计（按 ADAU1962A 重算）

竞品 ACM3128A 功放输入规格（已知确认值）：
- 输入灵敏度：**0.246 Vrms**（满功率输出）
- 输入阻抗：**20kΩ**

ADAU1962A 单端满幅输出估算（[待核实]）：约 **0.5 Vrms**（单端，须查 Datasheet 确认）

**衰减比计算（按 0.5 Vrms 单端满幅）**：
```
目标输出 Vout = 0.246 Vrms
DAC 单端满幅 Vdac ≈ 0.5 Vrms（[待核实]）
衰减比 K = Vout / Vdac ≈ 0.246 / 0.5 ≈ 0.492

分压网络设计（串联 R1，并联 R2 接地）：
  R2 / (R1 + R2) = 0.492
  → 取 R2 = 10kΩ（标准值），R1 = 10kΩ（实际 K = 0.500，偏差 +1.6%，可接受）
     或更贴近：R2 = 10kΩ, R1 = 10.2kΩ（1% 系列，K = 0.495，偏差 +0.6%）
  → 等效驱动阻抗 Zeq = R1 // R2 = 10k // 10k = 5.0kΩ << 竞品功放输入 20kΩ，负载效应小（√）

实际输出电压（R1=10k/R2=10k）：0.5 × 10/(10+10) = 0.250 Vrms（目标 0.246 Vrms，偏差 +1.6%，满足要求）
```

> **注意**：上式以 ADAU1962A 单端满幅 = 0.5 Vrms [待核实] 为前提。**实际 Datasheet 满幅一旦确认，须按真值重算 R1/R2**：
> - 通式：`R2/(R1+R2) = 0.246 / Vdac_fullscale`，固定 R2 = 10kΩ，则 `R1 = R2 × (Vdac/0.246 − 1)`。
> - 若 ADAU1962A 满幅为差分 ±1.0Vrms 且采用差分→单端转换，需在 I/V/缓冲级一并核算增益。

### 1.2 TDM 时钟方案（8 slot，与 DSP 对齐）

#### 时钟架构

```
EV-21569-EZKIT
  ├── 板载晶振：25MHz 工业晶振（竞品使用 ±50ppm；EZKIT 实际晶振规格[待核实]）
  ├── 21569 内部 PLL → 生成 MCLK = 12.288 MHz（48kHz × 256；或 24.576MHz 再分频）[待核实 PLL 配置]
  └── SPORT0 TDM 输出（8 通道，与 DSP 设计对齐）：
       - MCLK：12.288 MHz（主时钟，EZKIT 输出到转接板）[待核实 DAC 所需 MCLK 比率]
       - BCLK：12.288 MHz（8ch × 32bit × 48kHz = 12.288 Mbit/s）
       - FSYNC（LRCLK）：48 kHz
       - DATA：TDM-8（单条 DATA 线，8 时隙，每时隙 32bit）
```

> **BCLK 计算**：8 slot × 32 bit/slot × 48 kHz = **12.288 MHz**（v0.1 的 24.576MHz 对应 16 slot，已废止）。

#### 转接板时钟接收

| 信号 | EZKIT 输出 | ADAU1962A 接收 | 设计要点 |
|------|-----------|--------------|---------|
| MCLK | 12.288 MHz [待核实 DAC MCLK 比率] | 需 MCLK（[待核实]，部分 ADI DAC 内部 PLL 可从 BCLK 再生） | 串联 22Ω 阻尼电阻；走线 < 50mm |
| BCLK | 12.288 MHz | 直接接 BCLK pin | 串联 22Ω 阻尼；等长约束 |
| FSYNC | 48 kHz | 直接接 FSYNC/LRCLK pin | 串联 22Ω 阻尼 |
| DATA | TDM-8 串行 | 接 SDATA_IN | 8 slot 配置须与 DSP 一致 |

**TDM 配置强约束**：转接板 DAC 的 slot 数配置 = **8**，slot 宽 = 32bit，FSYNC 极性/相位、TDM 起始 slot 对齐方式必须与 21569 SPORT 输出严格一致。**调试首步：用逻辑分析仪确认 DAC 实际锁存的 slot 编号 = DSP 输出的 slot 编号**（防 1-slot 偏移导致全通道错位）。

**时钟质量要求**：
- EZKIT 时钟源抖动应 < 200 ps RMS（[估算]，音频 DAC 对抖动敏感，>1 ns 会引入明显噪声）
- 转接板时钟走线：阻控 50Ω 单端，长度 < 100mm，串联 22–33Ω 匹配电阻，不跨地割

### 1.3 情形分类设计（依 U1 拆机结果决定）

#### 情形①：竞品功放 = 模拟输入（如 ACM3128A 原厂模拟接口）

**适用场景**：确认竞品主板→功放板互连为模拟信号线。

转接板包含：
1. ADAU1962A（用 8ch DAC）+ 外围：I/V 转换/缓冲运放（建议 OPA1612 双通道 × 4 = 8 路，或 OPA4134 四通道 × 2）+ 重建 LPF（Butterworth 2阶，fc = 24kHz）（[待核实] ADAU1962A 是否内置输出缓冲，若有可省外部运放）
2. **8 路**电平/阻抗匹配衰减网络（R1/R2 分压，参数见 1.1 节，按 ADAU1962A 满幅重算）
3. 8 路 AC 耦合电容（薄膜电容，47nF，截止频率 @ 20kΩ负载 = 1/(2π×47nF×20kΩ) ≈ 170Hz）
4. 8 路限幅保护：串联 100Ω 电阻（限流）+ BAV99 或 PRTR5V0U2X 双向 TVS，钳位至 ±0.5 Vrms 余量（保护竞品功放输入级，不低于 ACM3128A 最大输入电压 [待核实]）
5. 上电静音电路：DAC MUTE 引脚由 RC 延迟（470kΩ × 10μF = 4.7s）控制，上电后 ~1s 解除 mute（防 pop）
6. **8 个逐通道测试点**（2.0mm 排针），支持单通道信号注入验证

**通道链路一致性（8ch）**：
ACM3128A 功放有 8 路输入（4片 × 2ch BTL）。竞品 A/B 串联后每路功放驱动 2 只喇叭（15Ω = 8Ω + 7Ω [估算]），16 只喇叭 = 8 功放路。
转接板输出的**模拟通道 = 8ch**（对应竞品 8 路功放输入），ADAU1962A 仅用 ch1~ch8（若实物 12ch，ch9~ch12 静默/不引出）。21569 软件配置 **TDM-8（8 时隙，全有效）**。

#### 情形②：竞品功放 = 数字 TDM 输入（最优捷径）

**适用场景**：拆机确认竞品主板→ACM3128A 功放板互连为 TDM 数字线。

```
EZKIT (21569) SPORT0 TDM-8 ──[电平缓冲/适配]──► ACM3128A 数字输入
```

**省掉 DAC、LPF、衰减网络**，转接板仅含：
- 逻辑电平转换（若竞品功放数字接口非 3.3V CMOS）：建议 TXS0108E（8位，双向，1.2V~3.6V）
- 时钟主从配置：建议 EZKIT/21569 做 BCLK/FSYNC 主，竞品功放做从
- ESD 保护（每条数字线串 33Ω + GND TVS）
- 数字隔离（可选，若地电位差 > 100mV）：ADuM1400（4路单向隔离，SI = 5kV）
- **TDM slot 数须确认竞品功放支持 8 slot（或可配）**，与 DSP 输出 8 slot 对齐 [待核实 U4]

**此情形复杂度大幅下降，优先验证。**

### 1.4 接地策略（数字/模拟隔离 + 单点共地）

**对齐竞品"数字地/模拟地隔离、单点共地"架构**：

```
转接板地分割：
┌─────────────────────────────────────────────────────┐
│ EZKIT 供电侧                                        │
│   DVDD 3.3V (数字)   AVDD 3.3V (模拟独立 LDO)      │
│        │                    │                       │
│       DGND              AGND (DAC模拟地)            │
│        │                    │                       │
│        └──────────┬─────────┘                       │
│               磁珠 (FB) 0Ω/单点连接                 │
│              (推荐 BLM21SN601, 600Ω@100MHz)          │
│               PGND (星型汇合点)                      │
│                    │                                │
│          ──────────┴──────────                      │
│         PCB 铜皮地面（完整地平面）                   │
└─────────────────────────────────────────────────────┘

EZKIT 与转接板共地：通过转接板连接器 GND 引脚单点连接
竞品功放供电：保留原供电，竞品 GND 通过信号线屏蔽/隔离
信号传输层：情形① → AC 耦合电容（隔直流，打断 DC 地环路）
            情形② → ADuM 数字隔离（可选，视地电位差决定）
```

**禁止**：EZKIT 给竞品功放供电（功率不够 + 强地耦合 + 供电污染）。

来源：EZKIT 扩展接口 +5V 或 USB 5V（[待核实] EZKIT 可提供的最大电流）

           │

## 3. 转接板关键设计约束汇总

| 约束类别 | 设计要求 | 来源 |
|----------|---------|------|
| **通道数/TDM** | **DAC 8 通道，TDM 8 slot / 32bit / FSYNC 48kHz / BCLK 12.288MHz，与 DSP 严格对齐**（F-X01 闭环） | DSP 8ch 设计 + ACM3128A 8ch |
| 电平匹配 | DAC 输出经衰减后 = 0.246 Vrms ± 10%（竞品 ACM3128A 满幅灵敏度）；R1/R2 按 ADAU1962A 实际满幅重算 | 竞品已知规格 + DAC Datasheet |
| 阻抗匹配 | 等效驱动阻抗 << 20kΩ（竞品输入阻抗）；设计 ≈5kΩ（√） | 竞品已知规格 |
| 过驱动保护 | 输出端钳位 ≤ ±[ACM3128A 最大输入电压 待核实] Vrms；串 100Ω 限流 | 防烧竞品功放 |
| 上电时序 | DAC 稳定后解除 MUTE（上电 ~1s 延迟）；先 EZKIT 后竞品功放 | 防 pop 保护 |
| 接地 | 数字地/模拟地单点汇接（磁珠或 0Ω）；转接板与竞品功放 AC 耦合隔直 | 噪声 / 嗡声抑制 |
| 通道可验证性 | 8 个独立测试点；支持单通道注入已知信号；逻辑分析仪确认 slot 对齐 | 调试映射验证 + 防 slot 偏移 |
| 法律合规 | 转接板仅内部使用；竞品器件按公开 datasheet 采购，不复制竞品 PCB/外壳/固件 | CLAUDE.md 法律红线 |
