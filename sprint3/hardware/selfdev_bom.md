## 0. 设计策略说明

**Sprint 3 长期自研路线**：复用竞品架构思路（合法借鉴），自主选型、自研 PCB。

竞品已验证架构（合法参考范围）：
- ADSP-21569 + DDR3 + SPI Flash boot
- 8ch D类功放 ACM3128A 驱动 16只喇叭（A/B串联，8路15Ω）
- 数字 TDM 协议互连
- 数字/模拟地隔离，单点共地
- 25MHz 工业晶振 ±50ppm

**自研差异化点**：
- DAC：自选（**ADAU1962A，8 通道方案**，对齐末端 8 驱动通道；A/B 串联后 16 只喇叭由 8 路功放驱动）
- 功放：评估保留 D类 ACM3128A-等效 vs 旧 BOM 的 TAS5825M（本文对比分析，见第 4 节）
- 电源树：自研设计，高集成、低噪声
- PCB layout：自研 4 层板，不复制竞品 PCB

## 1. 系统架构框图

```
外部供电（推荐 24V DC 适配器 or 明纬模块电源）
         │
    电源树（Buck + LDO，见第 5 节）
         │
    ┌────┴──────────────────────────────┐
    │                                   │
ADSP-21569KBCZ10（主 DSP）          功放电源轨（8-24V）
    │                                   │
    ├─ SPORT0 TDM-8 → ADAU1962A (用 8ch DAC) ─┤
    │   (8 slot/BCLK 12.288MHz/FSYNC 48kHz) │ 8路模拟输出
    ├─ DDR3 SDRAM（8Gb，外挂）          ACM3128A × 4（8ch D类功放）
    │                                         │
    ├─ SPI Flash（boot ROM）           8路BTL输出 × 2只/路
    │                                  = 16只喇叭（A/B串联，8路15Ω）
    ├─ I2C → 功放控制（ACM3128A 各片）
    │
    └─ JTAG/UART（调试接口）

全链路通道数 = 8：DSP(8 slot) → DAC(8ch) → 功放(8ch BTL) → 喇叭(8路×2只=16只)
时钟：25MHz 工业晶振 → ADSP-21569 内部 PLL → 12.288MHz MCLK（8ch×32bit×48kHz）[待核实 PLL 配置]
```

#### 5.2.2 D 类功放热耗散（ACM3128A × 4 = 8ch）

D类功放效率通常 85–92%（[估算]，ACM3128A 具体效率[待核实]）。

**典型语音场景（平均功率约额定的 10%）**：
- 每路输出：8Ω 喇叭，串联后 15Ω/路，额定输入 24V
  - 最大输出功率（单路 BTL @ 24V，8Ω 竞品参数）：[估算] P_max = (24/2)² / (2×8) ≈ 18W/路（BTL 近似）
  - 典型语音（10%平均功率）：≈ 1.8W/路 × 8路 = 14.4W 输出
  - 功放热耗散（η=88%）：14.4 × (1-0.88)/0.88 ≈ 2.0W [估算]

**最大正弦场景（额定满功率，1kHz 正弦）**：
- 8路满功率：约 18W/路 × 8路 = 144W 输出 [估算]
- 功放热耗散（η=88%）：144 × (1-0.88)/0.88 ≈ 19.6W [估算]
- **总功耗（功放段）= 输出 144W + 热耗 20W = 164W 输入** [估算]

## 6. 热设计估算

### 6.1 ADSP-21569 热设计

参数（[估算，待 ADI Datasheet 核实]）：
- 封装：400-ball CSP BGA，17×17mm
- Tj_max：125°C（[待核实]）
- Rjc：约 3–5°C/W（BGA 封装估算）[估算]
- 热设计功耗（TDP）：约 4–6W（[估算]，Sprint 2 skill.md 模板：5.5W TDP）

**密闭腔体场景热设计（沿用 Sprint 2 结论）**：

DSP 在密闭腔体中，无强制风冷时：
```
Tj = Ta + Pd × (Rjc + Rcs + Rsa)

假设：
  Ta = 45°C（最恶劣环境，设计目标）
  Pd = 5.5W（TDP）
  Rjc = 4°C/W（BGA 底部导热，[估算]）
  Rcs = 0.5°C/W（导热垫）
  Rsa = 散热片热阻

目标：Tj ≤ Tj_max × 0.8 = 125 × 0.8 = 100°C

所需散热路径热阻：
  Rsa ≤ (100 - 45) / 5.5 - 4 - 0.5 ≈ 5.5°C/W

→ 小型铝合金散热片 RSA ≈ 5°C/W（如 25×25×10mm 自然对流），
  密闭腔需额外开孔通风（有效通风面积 ≥ 20cm²，Sprint 2 结论）
```

**结论**：21569 在密闭腔体中**必须加小散热片（25×25×10mm 铝合金）+ 腔体开孔通风**，不建议无散热措施运行。与 Sprint 2 DEC-S2-005 热设计注意事项一致。

### 6.2 D 类功放（ACM3128A × 4）热设计

| 场景 | 每片功耗（热耗）| 封装 | 散热方案 | 备注 |
|------|--------------|------|---------|------|
| 典型语音 | ≈ 0.5W/片 [估算] | [待核实] ACM3128A 封装 | 铜皮散热（2oz 铜，≥4cm²/片）| Sprint 2 结论：靠铜皮散热可行 |
| 最大正弦 | ≈ 5W/片 [估算] | — | 铜皮 + 导热垫 + 铝底板 | 需实测验证温升 |

> [待核实] **ACM3128A 封装与热阻**：ACM3128A 封装信息（SOP/QFN/等）及 Rjc 需查厂商 Datasheet，决定最终散热方案。若为 QFN 底部热焊盘，铜皮散热效果好；若为 SOP，需加铝散热片。
