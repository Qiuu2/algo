## 3. 已验证的声学设计（数值经 PM 用 Balanis 闭式 + 数值积分独立复核）

阵列：16 阵元均匀线阵，间距 d=30 mm，孔径 L=(N−1)d=0.45 m，c=343 m/s，远场，均匀加权（基线）。

| 频率 | −3 dB 宽 | −6 dB 宽 | DI | 说明 |
|---|---|---|---|---|
| 500 Hz | 78.7° | 119.4° | 2.0 dB | 低于指向性频段下限 → 近似无指向性，**符合规格/预期** |
| 1 kHz | 37.0° | 51.2° | 4.8 dB | 指向性规格从此频点起生效 |
| **2 kHz** | 18.2° | **24.9°** ✅ | 7.6 dB | **设计频点 — 通过（≤30°，余量约 5°）** |
| 4 kHz | 9.1° | 12.4° | 10.5 dB | 强指向，第一旁瓣 −13.2 dB |

- **目标判定：** −6 dB 波束宽度 ≤ 30° @ 2 kHz → **PASS（24.9°，余量约 5°）**。
- **栅瓣：** 1–4 kHz 全频段无栅瓣；±30° 偏转起始 7,622 Hz，空间-Nyquist c/2d=5,717 Hz，宽边起始 c/d=11,433 Hz — 三个判据均 > 4 kHz，安全。
- **加窗权衡（2 kHz，Dolph-Chebyshev）：** 均匀 24.9°/−13 dB；**−20 dB → 26.8°（通过，余量 3.2°，推荐）**；−25 dB → 29.5°（零余量）；−30 dB → 31.9°（超标）。
- **低频说明（符合规格）：** 按 CTO 决策，指向性仅要求 ≥1 kHz；500 Hz 处孔径短于波长（L/λ=0.66）呈近似无指向性，为预期且可接受的产品行为，非缺陷。0.45 m 孔径对 1 kHz 下限是正确尺寸。

## 4. 已验证的 DSP / SHARC 算力可行性（数值经 PM 复核）

**关键结论：** 简单 DAS 无法实现跨频段恒定波束 → **filter-and-sum（FAS）为必需**；分数延迟 FIR 为必需（仅整数延迟会导致 ±6.8° 偏转误差）。

**FAS 设计（1–4 kHz，2 倍频程；±2 dB 恒定波束 → 4 子带）：**

| 方案 | 架构 | 算力 (16 ch) | SC589 单核（cap 402） | 21569（cap 892） | 结论 |
|---|---|---|---|---|---|
| **B（推荐）** | 4 子带多相抽取 FAS | **79.9 MMAC/s** | **19.9%** | **9.0%** | 可行 |
| A（保守） | 单级 283 抽头全速率 FIR/ch | 229.7 MMAC/s | 57.1% | 25.7% | 可行 |

- 算力预算规则：可用 = 总值 × 0.85；单算法 ≤ 可用 × 0.70。两方案均在预算内。
- 方案 A 算力对账：217.3（283 抽头 FIR）+ 11.5（3 个 biquad）+ 0.8（求和/归一化）= 229.6 MMAC/s，自洽。
- 内存约 15–20 KB（可用 ≥512 KB/核）；端到端延迟约 2.0 ms（预算 20 ms）。
- Lagrange 15 阶分数延迟 FIR 群延迟在 1–4 kHz（归一化 0.021–0.083）平坦，纹波 < 0.01 样本。
- SHARC+ 浮点原生，无需定点溢出防护。
- 两片候选芯片算力均大幅过剩；选型应按 I/O/功耗/成本（Gate 2）。
