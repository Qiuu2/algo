# 明天实验单 — 定位"波束为何形不成"(2026-07-01 台前可执行 · critic 过门定稿 v2)

> **【已闭合 2026-07-08 → 见 `BEAM_POLARITY_CLOSURE_20260708.md`】** 本单的**极性反接假设方向判对了**。根因坐实 = 物理 **C2、C4 两对喇叭 +/− 接反**(接竞品喇叭时接混)。实际定位用的是后来的 **375 静态逐路声学法(EXP_STATIC_POL375)**(本单的 C0/C2 近场声学法被静态法取代:静态法不受近场干涉/环境影响,更干净)。翻正 C2/C4 → M2 定向 2kHz 18dB。本单存史。

> **背景一句话**：整机 Test3(水平指向)实测乱(0° 不是最大、±90° 仅低 3-4dB、同位置抖)。两轮反查 workflow 收敛 + 竞品同场景 1m 出 ~12dB 对照 → **最可能根因 = 8 路模拟通道(DAC→功放→串联对→16 元)在正前方不相干,大概率某条串联对极性反接**。两种反接都要查:**(A) 整对共模反**(一路功放正负接反,两个镜像元件一起反)；**(B) 对内差分反**(一对 {c,15-c} 里**只反一只**喇叭)。DSP/算法侧已逐行亲核为干净同相 Dolph,数学上产不出这个现象。
> **明天目的**：用便宜实验**当场定位**是不是极性、是哪一路/哪一只；并实验性排除"测量/环境"。
> **来源**：defect-hunt workflow(wf_1ad922f8)+ PM numpy 近场复算 + **独立 critic 过门(claude-opus-4-8,CONDITIONAL→修 1 BLOCKER/3 MAJOR/4 MINOR→本 v2)**。**性质**：实验程序,非结论;无采购、无不可逆动作。
> **⚠ 近场口径**：所有 1m 读数都是**近场**(2kHz 远场≈8m),**不是远场指向性规格值**,只作**相对比较 / 竞品背靠背**用。健康阵列在 1m 正前仍是最大值,故"0° 不是最大"仍是有效的故障信号。

## 0. 总览(测试员先读这段)
沿信号链二分,每个实验只动一个变量。优先级:
1. **实验 C(首发,必做,只用分贝计)**：三小步 **C0 逐对 solo 指纹 → C2 配对 → C1 累加**,测 8 路在正前是否**相干叠加**,定位**哪一路/哪一只反相**,并顺带**核对通道↔对的映射**。
2. **实验 A(必做,若有竞品)**：竞品 vs 我们**背靠背**,把"测量/环境"一刀切。
3. **实验 B(加分,需信号源+分配器)**：绕过 DSP,判"物理链坏 vs M2 坏"。
4. **实验 D(加分,需示波器)**：把"极性错"和"功放相位失配"分开。

**最关键纪律(否则白测/误判)**：
- **实验 C 必须用 ~300Hz 低频(300–400Hz,500Hz 是硬上限,绝不用 2kHz)**——2kHz 在 1m 近场会让接对的外侧对也"掉",误判成反相 [L2 已核;500Hz 以上 c0 反相信号已 <3.3dB 不可靠]。
- **对内差分反相(只反一只)在 C2/C1 里看不出来**(在轴恰好抵消=贡献 0dB,和"该对没开"一样)→ **必须先做 C0 逐对 solo 指纹**才抓得到。这正是你们 DEC-S6-TEST1-METHOD-01 "改逐路 solo" 的同一招。
- **不要拿"声音抖动大小"当判据**——相干阵在 1m 近场也抖 ~10dB,抖动两可。判据锚在 **solo 指纹 / 正前累加 / front-to-side**。
- **拆任何线之前,先用 C0+C2(+D)确认是哪一路/哪一只,再动手**;改接线前功放断电。

## 1. 实验前准备

### 1.1 器材
| 等级 | 器材 | 用于 |
|---|---|---|
| **必备** | 分贝计(手机 App 可,**有平均/Leq/慢响应更好**)；能放 **300Hz / 2kHz / 宽带粉噪** 的信号源；卷尺或激光测距；能**逐路断开功放输入并贴标签**的接插方式 | 实验 C |
| **必备(若做A)** | **竞品音柱** + 同一支分贝计 | 实验 A |
| 加分 | 信号发生器 + **无源 1分8 分配器**(灌功放输入端) | 实验 B |
| 加分 | 示波器 + 触发；贴面麦或加速度计 | 实验 D |

### 1.2 开测前必查(前提,缺一不可)
1. **阵列横放**(长轴水平)、broadside 朝正前——保持不变。
2. **跑的是 M2** 不是 M1 透传:板上读 `g_m2_fira_inloop=1` / `g_m2_fg_beam_live=1` / `g_m2_out_nonzero` 在增长。**每次声学测量时同时复看这三个**。
3. **给 8 路功放输入接插口贴标签 c0..c7**(按设计:c7=中心对、c0=最外对)。这个标签对不对,C0 指纹会顺带验(见 C0)。
4. 信号放 300Hz 时**确认 300Hz 真能从喇叭出来**(耳朵听到低音)。
5. 功放**从最小音量/断电起步**,逐步上;刺耳/破音立即降。

### 1.3 安全
- 真喇叭驱动:功放最小音量起步;改接线前**功放断电**。
- 断开的功放输入若浮空会哼——把该路功放关掉或输入接地,确保"断开=真静音"。
- 无采购、无不可逆动作;异常**记录发回,不自行改算法/build**。

## 2. 实验

### ★ 实验 C — 极性/相干/映射(首发,必做,只用分贝计,300Hz)
**排查的因素**：8 路在正前是否同相相干叠加；**哪一路共模反相 / 哪一只对内反相**；通道↔对映射对不对。
**为什么 300Hz**：1m 外侧对程差 81mm,在 300Hz 只 26°(接对了仍明显涨),在 2kHz 是 171°(接对了也掉,会误判)[L2 已核]。
**摆位**：麦放**正前 0°、~1m、阵列中心高度**,**全程不动**。每个读数取 0° 附近相距 ~10cm 的 3 点中值(压近场散斑)。M2 跑 **300Hz**。

#### ▶ C0 逐对 solo 指纹(**必做第一步,抓"对内反相"+核映射**)
**只接通一对**(其余 7 路功放输入断开/静音),读 0° SPL;对 c0..c7 各做一次。**再各加测一个 ±45° 读数**(见判据②)。
**健康预期(相对 c7-solo,[L2 近场已核]）**——就是 Dolph 指纹(中心最响、c1 最弱、c0 抬边):
| 对 | c7 | c6 | c5 | c4 | c3 | c2 | c1 | c0 |
|---|---|---|---|---|---|---|---|---|
| 在轴 SPL(dB) | 0(基准) | −0.3 | −0.9 | −1.7 | −3.0 | −4.5 | **−6.5** | −1.9 |
**判据**：
- **① 抓对内差分反相**：某对在轴 SPL **比指纹深掉 / 近乎静音**(理论 −∞,实测被噪底兜在很低)→ **该对里一只喇叭反接**(对内差分反相)。当场记下 c。
- **② 哑路 vs 对内反相 区分**：对"近乎静音"的那对,加测 **±45°**:**离轴又明显响起来**(偶极子瓣,[L2]:如 c6 离轴 −10dB、c0 离轴 −1dB)→ 对内反相;**离轴也静音** → 该对开路/虚焊(哑路)。
- **③ 核映射 + 参照有效性**:8 路在轴电平排序应≈上表(c7 最响→c1 最弱→c0 抬边)。**排序乱 = 通道↔对映射接错**(锥化贴错位置),记录错位发回。**特别:中心对 c7 单独必须清晰可闻、明显高于噪底**——若 c7 单独就近乎静音,说明 c7 自己对内反相,**后面 C2 的参照失效,立即停下报回**。

#### ▶ C2 配对法(抓"整对共模反相",灵敏定位)
参照=**只接中心对 c7** 读 0° SPL(记 S7,须先在 C0 确认 c7 健康)。然后**每次只接 {c7 + 一个 ck}**(两对),读 0° SPL,对 c6..c0 各一次。
**预期(相对 S7,300Hz,[L2 含近场已核]）**：
| 开 | c7+c6 | c7+c5 | c7+c4 | c7+c3 | c7+c2 | c7+c1 | c7+c0 |
|---|---|---|---|---|---|---|---|
| **接对了(共模)** | +5.9 | +5.6 | +5.2 | +4.6 | +4.0 | **+3.3** | +4.9 |
| **整对共模反相** | −28.8 | −19.5 | −13.9 | −9.9 | −7.0 | **−4.8** | −7.1 |
| **对内差分反相**(C2 看不出) | **≈0(同"没开")** | ≈0 | ≈0 | ≈0 | ≈0 | ≈0 | ≈0 |
**判据**：
- **c7+ck 明显高于 S7(>+2dB)→ ck 共模接对了**；
- **c7+ck 低于 S7(尤其低 >3dB)→ ck 整对共模反相**(当场记下 c,断电查该路功放正负);
- **c7+ck ≈ S7(几乎没变)→ 不是共模问题,但可能是对内反相**(回看 C0 该对是否静音)。

#### ▶ C1 累加(仅作相干性快速概览,**判据弱,别据此拆线**)
只接 c7→逐对加回 c6→c0,每接一路读 0° SPL。理想同相应单调升,总升约 +16dB。
**[L2,远场理想化曲线;近场下 c4..c0 每步增量 <2.2dB,落在手机噪底内]**：
| 接到 | c7 | +c6 | +c5 | +c4 | +c3 | +c2 | +c1 | +c0 |
|---|---|---|---|---|---|---|---|---|
| 累计(dB) | 0 | +5.9 | +9.2 | +11.4 | +13.0 | +14.1 | +14.9 | +16.2 |
**判据(只信前两步)**：**+c6、+c5 两步必须是正增量**(各 +5.9 / +3.3);若这两步不升反降 → 该路反相。**c4 及以后增量太小(<2.2dB),在手机噪底内,不得据此判反相**——以 C0/C2 为准。

**实验 C 输出**：① 哪对对内反相(C0 静音+离轴响);② 哪路共模反相(C2 掉);③ 映射对不对(C0 指纹排序)。全部正常 → 极性/映射都对 → 转实验 D(查功放相位残余)。

### ★ 实验 A — 排除测量/环境:竞品 vs 我们 背靠背(必做,若有竞品)
**排查的因素**：坏结果是**环境/测量方法**,还是**我们系统**。
**步骤**：**同一支麦、同一增益、同一 1m、同一高度、同一信号(2kHz + 再跑一遍粉噪)、同一角度栅格 0/±30/±60/±90°**,竞品与我们前后脚各扫一遍。每角度取相距 10cm 的 3 点中值。竞品朝向/高度用卷尺对齐。
**判据**：
- 竞品 **≥10dB 稳定 front-to-side、我们 <6dB** → **环境无罪,缺陷在我们系统**(预期);
- 竞品在我们设置下**也塌** → **测量/方法**问题 → 回查麦位/增益/近场协议;
- **竞品与我们都落在 6–10dB 之间 = 不决定性**(部分退化)→ 不下结论,继续靠 C0/C2 定位。
**纪律**：用**多点平均角度方向图**比,不是单点抖动;竞品锥化可能不同,只取"有没有清晰波束"的定性结论。

---

### 实验 B — 物理链 vs M2:均匀并联驱动(加分,需信号源+分配器)
**排查的因素**：缺陷在**功放输入下游物理链**,还是 **M2/DSP/DAC**。
**步骤**：2kHz(再跑粉噪)经分配器**并联灌到 8 个功放【输入端,线电平,不是喇叭端!】**,8 路同相等幅=均匀 broadside,**绕过 DSP/M2/DAC**。测 0/±30/±60/±90°(多点中值)。
**判据(近场,已核)**：
- **0° 清晰是最大值,且 ±90° 比 0° 低 ≥~10dB**(健康均匀阵 1m 近场 L0−L90≈15.8dB [L2];**不要用 13-15dB 当门**——均匀阵中间旁瓣只低 ~6dB 是正常的,会误判)→ 功放输入下游物理链好 → 缺陷在 **DAC 通道路径 或 M2**(由 C0 的指纹映射核 DAC↔对那半边;DAC 单通道死/相位 留作后续板侧实验 F/E);
- **仍平**(0° 非最大 / ±90° 仅低几 dB)→ 缺陷在功放输入下游物理链 → 回 C0/C2/D 定位。
**陷阱**：① 必灌**输入端**,误灌喇叭端引入阻抗错;② 均匀驱动副瓣升到 ~−13dB,判据对**均匀阵**不是 Dolph;③ **对内差分反相的对在均匀驱动下仍抵消、仍是偶极子** → B"仍平"**不能区分**声道反 vs 对内反 vs 功放相位,只证"缺陷在 amp 下游、非 M2",定位仍靠 C0/C2/D。

---

### 实验 D — 极性 vs 功放相位 的终判:示波器门控首达(加分,需示波器)
**排查的因素**：把**极性错**(符号反)和**功放群延迟/相位失配**(符号对、时刻偏)分开——**唯一能分开这俩的实验**,C 对功放相位失明。
**步骤**：对每个喇叭单元喂门控 click/tone-burst,贴面麦/加速度计在**反射到达之前**读首达波的**符号(+/−)AND 相对到达时刻**,逐一记 16 个物理位置 pos0..15。低频段首达沿判极性最稳。
**判据**：
- 16 元首达**同号且时刻齐** → 极性+时序全对;
- 某镜像对 {posc,pos15-c} 两元**反号** → **对内差分反相**;整对相对其余对反号 → **共模反相**;
- 某通道**符号对但时刻系统性偏移 >~0.5 样本(≈10µs@2kHz≈7°)** → **功放群延迟失配**,非极性。

## 3. 决策树
```
实验C0(逐对solo指纹, 必做第一)
  ├ 某对在轴近乎静音 + 离轴又响 → 对内差分反相, 定位(记c) → 断电查该对焊点正负
  ├ 某对在轴静音 + 离轴也静音 → 哑路/开路 → 查该路功放+焊点
  ├ 8路在轴电平排序乱(≠Dolph指纹) → 通道↔对映射接错 → 查接线
  ├ 中心对c7单独就近乎静音 → c7自己对内反相, C2参照失效 → 停, 报回
  └ 指纹正常 → 进C2
实验C2(c7+ck配对)
  ├ 某c7+ck 低于S7(>3dB) → 该路整对共模反相, 定位 → 断电查该路功放正负
  ├ 某c7+ck ≈ S7(没变) → 回看C0该对(可能对内反相)
  └ 都明显高于S7(>+2dB) → 共模极性都对 → 实验D(查功放相位残余)
实验A(竞品背靠背) → 竞品≥10稳+我们<6=我们缺陷 / 都6-10=部分退化继续C / 竞品也塌=测量问题
实验B(均匀并联) → 0°清晰最大+±90≥~10dB=物理OK(问题在DAC路径/M2, 后续E/F) / 仍平=物理坏回C0/C2/D
实验D(示波器) → 符号反=极性 / 符号对时刻偏=功放相位失配
```
**今天最少做到**：实验 C(C0→C2→C1)+ 实验 A。两个**只用分贝计**就能出结论并定位。

## 5. 红线纪律(再强调)
1. **实验 C 用 300–400Hz(≤500Hz),绝不用 2kHz**(否则误判外侧对)。

5. **不证伪"测量无罪"前(实验 A)不下"算法坏"结论**。
