## 2. 根因与修复

- **根因**:物理 **C2、C4** 两对喇叭的 +/− 接反(接竞品喇叭时不知其正负约定接混)。混极性把正前相干叠加拆掉 → 波束打散成近全向"平"。
- **修复**:调换 C2、C4 这两对喇叭接线 +/−(整对翻)。定位 mask:0x40=C2 对、0x01=C4 对。
- **验证**:成对复测 C7+C2 / C7+C4 由 71dB→~93.5dB(转相长);静态 2250 出指向;M2 活波束 18dB@2k / 9dB@1k。

## 3. 与 07-01 交换实验的关系(铁律四/五:不过度调和)

**今日确证(强,受控 A/B)**:07-07 静态平 → 07-08 定向,**其间唯一改动 = C2/C4 喇叭 +/− 对调,板/固件/build 没动** → 这一步干净受控实验证明:**在我们自己这套 rig 上、把 C2/C4 接对后,板 + M2 算法能成 18dB 波束**。→ **板/算法有能力成波束,非当前 rig 的瓶颈。**

**但不能据此说"07-01 交换实验错了/是极性混淆"(critic MAJOR-2 纠正我早先的过度声明)**:
- 07-01 是**竞品整套 rig(竞品功放+竞品喇叭+竞品接线,极性本来是对的)只换板**,当时我们板平、竞品板 16dB[L1]。
- **C2/C4 反接是后来我们把竞品喇叭接到自己功放时、我们自己接错引入的,07-01 那套竞品 rig 上并不存在**(若存在,竞品板也会平,但它 16dB)。→ **今日的极性根因,逻辑上解释不了 07-01 的"板 vs 板"差异。**
- 因此诚实结论:**07-01 那次"我们板在竞品 rig 上平"至今没有解释**(可能当时板/build 有问题、之后变了,也可能那次实验另有未控因素,从现有记录**无法重构**)。**"板平反"只限于"在我们自己 rig、极性接对后板能成 18dB 波束"这个范围,不外推为"07-01 归因错了"。**
- 记忆条 test3-diagnosis-state 里"7-01 交换实验错误归因"的措辞同样过重,待一并订正为本节口径。

**教训(存档)**:①对照实验只有"没换的那些"真被固定才成立;②**更要防"急着把旧结论调和进新发现"——我早先把 07-01 说成"极性混淆的错误归因"本身就是一次过度声明,被独立 critic 拦下**。两次同一个病根:抢着给一个干净解释。关联 [[working-style-lessons]]。
