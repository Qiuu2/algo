### 1.1 单位存疑（诚实标注，不臆断）
- **Mmd 750.929u / Mms 782.750m**：面板单位列分别疑为 Kg / g。物理上 2 寸全频驱动 Mms ≈ 0.78 g 合理；
  若 Mmd=750.929 µg（0.00075 g）则与 Mms 0.78g 量级矛盾。**最可能口径**：Mms=782.750 mg=0.78283 g，
  Mmd=750.929 mg=0.75093 g（µ 与 m 前缀的截图判读须 critic/lead 三方再核；**数字逐位已三方一致，单位前缀待定**）。
- **Vas 46.001m「Gal」**：LEAP 常用 L（升）或 ft³；46.001 m³ 不合理（太大），46.001 mL=0.046L 偏小，
  46.001 L 对小驱动偏大。**单位口径待核**；不进任何强约束直到单位确认。
- **红线**：以上单位歧义**仅影响派生电声计算**（SPL 重做须用对单位）；**不影响 §3 的冲突/解锁结论**
  （SPLo 82.161 dB / BL 2.030 / Fs 236 / Qts 1.463 数值与单位均清晰）。

### 1.2 自洽交叉核（铁律七，第二独立核）
- Qts 核：Qms·Qes/(Qms+Qes) = 4.625×2.140/6.765 = **1.4632** vs 面板 Qts **1.463** ✓ 自洽。
- Kxm 3.770m == Levc 3.770m ✓（半感抗模型 X 项 = 音圈电感，LEAP 口径一致）。
- 这两项自洽 → 面板数值内部一致，支持 L1 类判读（非乱填/截图错位）。

## 2. SPL vs Freq 曲线（LMS 截图，独立特征化）

> 纵轴 dBSPL（左，0–120）+ Deg（右相位，–180..180）；横轴 20 Hz–20 kHz 对数。红=幅频。LMS 工具。
> **条件未标 caveat**：测量距离/驱动电压/障板（自由场 vs IEC 障板）/平滑（1/n oct）**截图未注** → 绝对 dB 值
> 仅作 **shape/相对** 用，绝对电平须待条件确认（不可直接当 1W/1m）。**[L1/仪器，条件 caveat → 形状 L1 / 绝对电平 L2-条件未定]**

**形状特征（dsp 独立读）**：
- **20–100 Hz**：~72–73 dB 起，30–100 Hz 浅 shelf ~75–80 dB（Fs≈236Hz 以下，低频滚降区）。
- **100–300 Hz**：~100 Hz 处台阶上跳，**ripple 密集 85–100 dB 带**，峰 ~100 dB 近 150–200 Hz（多个尖峰=
  驱动/测量环境共振 ripple）。
- **300 Hz–2 kHz**：~85–95 dB **锯齿 plateau**（工作主带，ripple 数 dB）。
- **~4.5–5 kHz**：**breakup 峰 ~102 dB**（全曲最高，振膜分割振动 breakup）。
- **5–20 kHz**：85–100 dB ripple，~12–15 kHz 次峰 ~95–98 dB，向 20 kHz 滚降。

## 3. 冲突 + 解锁分析（load-bearing）

### 3a. SPLo 82.161 dB [L1] vs 铭牌 88 dB [L4] — ~5.8 dB 冲突（铁律四类）
- 铭牌 88 dB SPL/2.83V/1m（=1W/1m@8Ω，full_teardown_v2.md:45）[L4 datasheet/标称]。
- LEAP 实测 SPLo **82.161 dB** [L1/仪器]。**冲突 ~5.8 dB，实测低于铭牌**。
- **铁律四（L1 与 LOCKED/标称冲突）**：测量值 [L1] 压过标称 [L4]。
- **DEC-S3-DSP-05（SPL 预测降级，缺 BL/Re/Fs/Qts）现 UNLOCKABLE**：BL=2.030 / Revc=7.600 / Fs=235.907 /
  Qts=1.463 **现已 [L1] 到齐** → SPL 电声模型重做的前置数据**解锁**。
  **方向预判（不在本任务重做）**：实测 SPLo 82.161 < 铭牌 88 < 旧占位 117dB → SPL 重做**极可能落更低**
  （比旧 117dB[L4 占位]/铭牌 88dB 都低）。**本任务仅登记为解锁工作项，CTO 排期（声学/硬件线）；不重做 SPL。**

### 3b. Re：LEAP Revc 7.600 vs DC 7.4Ω [L1 拆机] — 一致（说明为何）
- DC 7.4Ω = 万用表直流测量（full_teardown_v2.md:47）[L1]。LEAP Revc 7.600 = LEAP 由阻抗扫描拟合的音圈直流电阻。
- **一致（差 0.2Ω，2.7%）**：两者都是音圈直流电阻 Re，测量类略不同（万用表 DC 端点 vs LEAP 扫描拟合外推到 DC），
  ~3% 偏差在仪器间正常。**互证 → Re≈7.5Ω 双轨坐实**（铁律七双轨核）。full_teardown_v2:49 的「Re 估算 ≈7.0Ω」
  现被 LEAP 7.600 + DC 7.4 双 [L1] 取代。

### 3c. 限幅阈值（DEC-S5-EQ-O1-01 T_k=T·w_k）— 解锁什么 / 仍缺什么（诚实拆）
- **T/S 解锁**：电压域/激励域建模的电声参数 **BL=2.030 / Cms=581.483µ / Mms≈0.78g / Fs=236 / Re≈7.5** 现 [L1] →
  可建**位移(excursion)-电压**传递（给定输入电压 → 振膜位移），是限幅阈值按权重缩放 T_k=T·w_k 的电声基础。
- **仍缺（HARD，不可由本报告关）**：
  - **额定功率（thermal rated power）= Q-② 仍 vendor-pending**（含追问回复:20「额定 10W 预估…待厂家实测」[L4]）。
    T/S 报告**无热额定**（LEAP 面板不给热功率）→ 防烧的**热阈值**仍缺。
  - **Xmax（最大线性位移）**：**LEAP 这张截图无 Xmax**（面板未列）→ **位移限仍缺**。无 Xmax 则「灌多少电压
    撞到位移上限」无法定 → T_k 的**绝对值**仍不可锁，只能定**相对权重缩放结构**。
  - **诚实结论**：限幅器**结构**（per-channel + T_k=T·w_k 权重缩放，DEC-S5-EQ-O1-01）不变；**绝对阈值**仍待
    Xmax + 额定功率（Q-② vendor + Xmax 实测/datasheet）。**不可过度关闭 Q-②/Q-③。**

### 3d. EQ band 数（R3 问题）— driver-level 预览 / system-level 仍待 R3
- driver-level SPL 形状现可见：**100–300Hz ripple + ~4.7–5k breakup 峰 ~102dB**。
- **对 O1 的 2–3 biquad [L2-preview]**：driver 本身有 (i) 100–300Hz 起伏 (ii) 4.7–5k breakup 峰 →
  若要平整工作带，2–3 biquad 可处理 ~1 个 breakup 峰削峰 + ~1 个低中频 shelf/notch；**与 O1 的 2–3 biquad 量级
  相容（L2 预览，非锁定）**。
- **driver-level ≠ system-level（为何仍待 R3）**：本曲线 = **单驱动**（自由场/障板未注）。产品是 **16 元线阵 +
  音柱箱体 + broadside 波束**——阵列干涉、箱体谐振、波束频响整形会**改变系统正轴向响应**，与单驱动曲线不同。
  EQ 是补**系统**响应不是单驱动 → **band 数终定仍须 R3 系统级消声室实测**（DEC-S5-EQ-O1-01「band 数待 R3」不变）。

### 3e. v1 HF 聚焦带（≥4kHz）— driver HF 能力 note（flag only）
- v1 聚焦有用带 ≥4kHz（DEC-S5-V1-SCOPE-01）。driver 在 4–6k 有能力（breakup 峰 ~102dB @ ~4.7–5k 说明 HF 有输出），
  但 **breakup 区响应不平 + 指向性在 breakup 频段会乱**（振膜分割振动 → 离轴恶化）。
- **flag only**：4–6k 的「聚焦指向性 × breakup 交互」是 **R3/acoustic 问题**（声学 teammate + 消声室离轴），
  非本任务。登记：v1 HF 聚焦带正落在 driver breakup 区附近 → R3 须查 breakup 对波束/聚焦的影响。

### 3f. Fs 236 Hz vs 工作带 500Hz–6k — driver 全程在谐振之上（note）
- Fs=235.907 Hz < 工作带下限 500 Hz → driver **全工作带都在谐振频率之上**（质量控制区）。
  这是全频小驱动的正常工作点；低频（<Fs）滚降（曲线 <100Hz ~72–80dB 印证）不在我方工作带，无碍。note only。
